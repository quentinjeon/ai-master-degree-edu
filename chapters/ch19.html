Lagrangian과 KKT 조건

제약이 있는 최적화 문제. SVM, 정규화의 수학적 기반인 KKT 조건을 이해합니다.

🧒 초등학생 한 줄 설명

"하루 칼로리는 2000kcal 이하여야 한다"는 제약 아래 가장 맛있는 식사를 고르는 것 — 이것이 제약 최적화입니다.

  • ① 제약 없는 최적화: 무엇이든 먹고 싶은 대로 → 단순히 \(\min f(x)\)를 풀면 됨
  • ② 등호 제약 \(g(x) = 0\): "칼로리를 정확히 2000kcal 써야 한다" — 라인 위에서만 움직일 수 있음
  • ③ AI에서: SVM은 마진 최대화 + KKT 조건으로 서포트 벡터를 정의하고, L2 정규화는 Gaussian prior + KKT의 등가 관계로 이해할 수 있음
🔗

레벨 1·2 연결: Ch.18에서 배운 볼록 함수의 성질이 여기서 결정적입니다. 볼록 문제에서는 KKT 조건이 최적성의 필요충분조건이 됩니다. 이 챕터가 SVM(Ch.09), 정규화(Ch.08), 강화학습의 Policy gradient까지 연결됩니다.

🔒제약 최적화 문제의 표준 형식

제약 최적화 문제는 목적함수를 최소화하되, 등호 제약과 부등호 제약을 동시에 만족해야 합니다. AI 논문에서 이 형식이 반복적으로 등장합니다.

제약 최적화 표준형 (Standard Form)
\[ \begin{aligned} &\min_{x} \; f(x) \\ &\text{s.t.} \quad g_i(x) = 0, \quad i = 1, \ldots, m \\ &\phantom{\text{s.t.}} \quad h_j(x) \leq 0, \quad j = 1, \ldots, p \end{aligned} \]
\(f(x)\): 최소화할 목적함수 (예: 오차, 손실)
\(g_i(x) = 0\): 등호 제약 — 정확히 만족해야 하는 조건 (예: 확률의 합 = 1)
\(h_j(x) \leq 0\): 부등호 제약 — 넘어서는 안 되는 조건 (예: 가중치 크기 제한)
\(\text{s.t.}\): "subject to"의 약자 — "~를 만족하는 조건 아래"

AI 논문에서 제약 최적화 예시

SVM (소프트 마진)마진 역수 \(\frac{1}{2}\|w\|^2\)부등호: \(y_i(w^\top x_i + b) \geq 1\)서포트 벡터 정의

AI 논문에서 "feasible region (가능 집합)"이란 모든 제약을 만족하는 \(x\)의 집합입니다. 우리는 이 집합 안에서 \(f(x)\)를 최소화해야 합니다. Ch.18에서 배운 볼록 집합이 가능 집합인 경우, 전역 최솟값이 존재하고 KKT로 찾을 수 있습니다.

ℒLagrangian — 제약을 목적함수 안으로 집어넣기

Lagrangian은 제약 조건을 목적함수 안에 "패널티"처럼 합산한 함수입니다. 제약을 외부에서 강제하는 대신, Lagrange 승수(\(\lambda, \mu\))를 도입해 목적함수와 합산합니다. 이렇게 하면 제약 문제가 무제약 새들포인트(saddle-point) 문제로 바뀝니다.

Lagrangian 함수 (Lagrangian Function)
\[ \mathcal{L}(x, \lambda, \mu) = f(x) + \sum_{i=1}^{m} \lambda_i g_i(x) + \sum_{j=1}^{p} \mu_j h_j(x) \]
\(f(x)\): 원래의 목적함수
\(\lambda_i\): 등호 제약 \(g_i(x) = 0\)에 대응하는 Lagrange 승수 — 부호 제한 없음 (\(\lambda_i \in \mathbb{R}\))
\(\mu_j\): 부등호 제약 \(h_j(x) \leq 0\)에 대응하는 Lagrange 승수 — 반드시 \(\mu_j \geq 0\) (쌍대 가능성)
\(\lambda_i g_i(x)\): 등호를 어기면 패널티가 무한히 커짐
\(\mu_j h_j(x)\): \(h_j(x) > 0\)이면 \(\mu_j \geq 0\)이므로 패널티 증가 → 가능 집합 밖으로 못 나감
🔑 Lagrangian의 새들포인트 해석

원래 문제의 해 \(x^*\)는 Lagrangian의 새들포인트(saddle point)입니다:

\[ \mathcal{L}(x^*, \lambda, \mu) \leq \mathcal{L}(x^*, \lambda^*, \mu^*) \leq \mathcal{L}(x, \lambda^*, \mu^*) \]

즉 \(x^*\)는 \(\mathcal{L}\)을 \(x\)에 대해 최솟값으로 만들고, \((\lambda^*, \mu^*)\)는 최댓값으로 만듭니다. 이 기묘한 구조가 프라이멀·쌍대 이론의 핵심입니다.

쌍대 문제 (Dual Problem)

Lagrangian에서 \(x\)를 먼저 최소화하면 쌍대 함수 \(d(\lambda, \mu) = \min_x \mathcal{L}(x, \lambda, \mu)\)를 얻습니다. 강한 쌍대성(strong duality)이 성립하면 \(\min_x f(x) = \max_{\lambda, \mu \geq 0} d(\lambda, \mu)\)입니다.

프라이멀 문제 (Primal)
  • \(\min_x f(x)\) s.t. 제약
  • \(x\)가 변수, \(\lambda, \mu\)는 고정
  • 가능 집합 안에서 최솟값 탐색
  • SVM에서: 마진 최소화 형태
쌍대 문제 (Dual)
  • \(\max_{\lambda, \mu \geq 0} d(\lambda, \mu)\)
  • \(\lambda, \mu\)가 변수, \(x\)는 제거됨
  • 쌍대 가능 집합에서 최댓값 탐색
  • SVM에서: 서포트 벡터만으로 표현

✓KKT 조건 — 최적해의 필요충분조건

KKT(Karush-Kuhn-Tucker) 조건은 제약 최적화 문제의 최적해 \(x^*\)가 만족해야 하는 4가지 조건입니다. 볼록 문제에서는 이 조건이 필요충분조건이 됩니다. SVM, 정규화, 강화학습 등 거의 모든 AI 최적화 이론의 핵심입니다.

조건 1
정류 조건 (Stationarity)

\(\nabla_x \mathcal{L}(x^*, \lambda^*, \mu^*) = 0\) — Lagrangian의 \(x\)에 대한 기울기가 0

조건 2
가능성 (Primal Feasibility)

\(g_i(x^*) = 0\), \(h_j(x^*) \leq 0\) — 원래 제약을 모두 만족

조건 3
쌍대 가능성 (Dual Feasibility)

\(\mu_j^* \geq 0\) — 부등호 제약의 승수는 반드시 비음수

조건 4
상보 여유 (Complementary Slackness)

\(\mu_j^* \cdot h_j(x^*) = 0\) — 제약이 활성화되지 않으면 승수는 0

KKT 조건 (4가지 — 볼록 문제에서 필요충분)
\[ \begin{aligned} (1)\; &\nabla_x \mathcal{L}(x^*, \lambda^*, \mu^*) = 0 \quad &\text{(정류 조건)}\\ (2)\; &g_i(x^*) = 0, \quad h_j(x^*) \leq 0 \quad &\text{(가능성)}\\ (3)\; &\mu_j^* \geq 0 \quad &\text{(쌍대 가능성)}\\ (4)\; &\mu_j^* \cdot h_j(x^*) = 0 \quad &\text{(상보 여유)} \end{aligned} \]
조건 (1): 최적점에서 Lagrangian의 기울기가 0 — 경사하강법으로 더 내려갈 수 없는 상태
조건 (2): 최적해는 반드시 원래 제약을 만족해야 함 (가능 집합 안에 있어야 함)
조건 (3): 부등호 제약의 승수 \(\mu_j^* \geq 0\) — 부호를 바꾸면 제약 방향이 반대가 됨
조건 (4): \(\mu_j^* > 0\)이면 \(h_j(x^*) = 0\) (활성 제약), \(h_j(x^*) < 0\)이면 \(\mu_j^* = 0\) — 둘 중 하나는 반드시 0

상보 여유 조건의 의미 — SVM 서포트 벡터

상보 여유 조건 \(\mu_j^* \cdot h_j(x^*) = 0\)은 두 가지 경우만 허용합니다:

경우 A: 비활성 제약
  • \(h_j(x^*) < 0\) — 제약을 여유롭게 만족
  • \(\Rightarrow \mu_j^* = 0\) — 승수는 자동으로 0
  • 이 제약은 최적해에 영향 없음
  • SVM: 마진 안쪽 데이터 → 계수 = 0
경우 B: 활성 제약
  • \(h_j(x^*) = 0\) — 제약 경계에 정확히 위치
  • \(\Rightarrow \mu_j^* \geq 0\) — 승수가 양수일 수 있음
  • 이 제약이 최적해를 결정
  • SVM: 마진 경계 데이터 = 서포트 벡터
💡

SVM의 서포트 벡터는 KKT 상보 여유의 결과입니다. SVM에서 제약은 \(h_i(w, b) = 1 - y_i(w^\top x_i + b) \leq 0\). 상보 여유 \(\mu_i^* \cdot h_i = 0\)에 의해, \(\mu_i^* > 0\)인 데이터만이 결정 경계에 기여합니다. 이것이 바로 마진 위에 위치한 "서포트 벡터"의 수학적 정의입니다.

🔗L2 정규화와 KKT — 등가 관계

L2 정규화(Ridge)는 단순히 "오차 패널티"가 아닙니다. KKT 이론을 통해 제약 최적화 문제와 완전히 등가임을 보일 수 있습니다. 이 등가 관계는 정규화 강도 \(\lambda\)와 제약 크기 \(t\) 사이의 일대일 대응을 보여줍니다.

L2 정규화 ↔ 제약 최적화 등가 관계
\[ \underbrace{\min_\theta \; \mathcal{L}(\theta) + \frac{\lambda}{2}\|\theta\|^2}_{\text{Lagrangian 형태}} \quad \Longleftrightarrow \quad \underbrace{\min_\theta \; \mathcal{L}(\theta) \;\; \text{s.t.} \;\; \|\theta\|^2 \leq t}_{\text{제약 최적화 형태}} \]
\(\mathcal{L}(\theta)\): 원래 손실함수 (예: 교차 엔트로피, MSE)
\(\lambda\): 정규화 강도 (Lagrange 승수 역할) — \(\lambda\)가 크면 \(t\)가 작음
\(\|\theta\|^2 \leq t\): 파라미터 노름의 상한 제약 — 과적합 방지
좌변에서 \(\lambda\)는 고정값, 우변에서 \(t\)는 제약의 크기
둘 중 하나의 KKT 조건을 풀면 나머지의 해도 자동으로 구해짐
🔑 Bayesian 관점과의 연결

L2 정규화의 Bayesian 해석: \(\theta\)에 Gaussian prior \(\theta \sim \mathcal{N}(0, \sigma^2 I)\)를 부과한 MAP 추정과 동일합니다.

\[ \underbrace{\min_\theta \left[\mathcal{L}(\theta) + \frac{\lambda}{2}\|\theta\|^2\right]}_{\text{L2 정규화}} = \underbrace{\max_\theta \left[\p(\text{Data}|\theta) \cdot \mathcal{N}(\theta; 0, \sigma^2 I)\right]}_{\text{MAP 추정}} \]

여기서 \(\lambda = 1/\sigma^2\). 정규화 강도가 클수록 prior의 분산이 작아져 파라미터가 0 근방에 집중됩니다. KKT는 이 관계를 제약 최적화 언어로도 표현할 수 있게 해줍니다.
