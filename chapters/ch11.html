정보이론: Entropy, Cross-Entropy, KL Divergence

딥러닝 손실함수의 수학적 근거. 불확실성을 정량화하고 두 분포의 차이를 측정합니다.

🧒 초등학생 한 줄 설명

"Entropy는 '얼마나 헷갈리는지'. Cross-Entropy는 '정답과 AI 예측이 얼마나 다른지 재는 벌점'. KL Divergence는 '두 생각이 얼마나 다른지'."

  • Entropy: 동전이 앞/뒤 반반 → 제일 헷갈림 → H 최대. 항상 앞면 → 전혀 헷갈리지 않음 → H = 0
  • Cross-Entropy: AI가 "고양이 99%"라고 했는데 정답이 고양이 → 벌점 작음. "고양이 5%"인데 정답이 고양이 → 벌점 큼
  • AI에서: 딥러닝 분류 학습 = Cross-Entropy 손실 최소화 = KL(P‖Q) 최소화

🌀엔트로피 (Entropy) — 불확실성의 척도

Shannon Entropy는 확률분포의 불확실성(정보량)을 측정합니다. 분포가 한 값에 집중될수록 엔트로피는 낮아지고, 모든 값이 균등하게 분포될수록 엔트로피는 높아집니다.

Shannon Entropy
\[ H(P) = -\sum_{x} P(x) \log_2 P(x) \ge 0 \]
\(H(P)\): 분포 P의 Shannon Entropy — 불확실성의 정량적 척도
\(\log_2\) 사용 시 단위는 "bits" (밑이 \(e\)이면 "nats")
\(P(x)=0\)인 항은 0으로 정의: \(\lim_{p \to 0} p \log p = 0\)
균등분포일 때 최대: \(H(\text{uniform on } K) = \log_2 K\) bits
확실할수록 (\(P=1\)인 값이 있으면) \(H = 0\) — 불확실성이 없음
분포H 값의미
[1.0, 0.0]0 bits완전히 확실
[0.5, 0.5]1 bit최대 불확실 (동전)
[0.9, 0.1]0.469 bits어느 정도 확실
균등 (K 클래스)\(\log_2 K\) bitsK가 클수록 불확실

🏹교차 엔트로피 (Cross-Entropy)

정답 분포 P를 모델 분포 Q로 인코딩할 때 필요한 평균 비트 수입니다. 딥러닝 분류 학습의 손실함수로 사용되며, P에 맞는 최적 코드 대신 Q에 맞춘 코드를 사용할 때 발생하는 추가 비용을 측정합니다.

Cross-Entropy
\[ H(P, Q) = -\sum_{x} P(x) \log Q(x) = H(P) + D_{KL}(P \| Q) \]
\(P\): 정답 분포 (one-hot label) — 실제 정답이 나타내는 확률 분포
\(Q\): 모델의 예측 분포 (softmax 출력) — AI가 추정하는 확률 분포
\(H(P)\): 정답 자체의 불확실성 — 학습으로 줄일 수 없는 고정된 양
\(D_{KL}(P\|Q)\): P와 Q의 차이 — 학습을 통해 줄여나가는 부분
P가 one-hot이면 \(H(P)=0\)이므로: \(H(P,Q) = D_{KL}(P\|Q)\)

↔️KL 발산 (Kullback-Leibler Divergence)

KL Divergence는 두 확률분포 P와 Q 사이의 "정보 손실"을 측정합니다. P를 Q로 근사할 때 발생하는 추가 정보량을 나타내며, 항상 0 이상입니다. 단, 대칭적이지 않아 거리(metric)는 아닙니다.

KL Divergence (비대칭 거리)
\[ D_{KL}(P \| Q) = \sum_x P(x) \log \frac{P(x)}{Q(x)} = H(P,Q) - H(P) \ge 0 \]
항상 \(\ge 0\): Gibbs 부등식에 의해 보장
\(P = Q\)일 때만 \(D_{KL} = 0\) — 두 분포가 완전히 같을 때만 0
비대칭: \(D_{KL}(P\|Q) \neq D_{KL}(Q\|P)\) — 순서가 중요
"P를 Q로 근사할 때의 정보 손실" — Q가 P를 얼마나 잘 표현하는지의 척도
개념수식의미AI 역할
Entropy \(H(P)\)\(-\sum P \log P\)P 자체의 불확실성정보량 측정, 결정트리
Cross-Entropy \(H(P,Q)\)\(-\sum P \log Q\)Q로 P를 표현할 때 비용분류 손실함수
KL Divergence \(D_{KL}(P\|Q)\)\(\sum P \log(P/Q)\)P와 Q의 차이VAE, RLHF, 지식증류
💡

\(H(P,Q) = H(P) + D_{KL}(P\|Q)\). 정답 레이블이 one-hot이면 \(H(P)=0\)이므로 CE = KL.
따라서 CE 최소화 = KL 최소화 = 두 분포를 최대한 비슷하게 만들기

📐Jensen's Inequality — KL 비음수성의 증명 핵심

정보이론 논문에서 KL 발산이 항상 0 이상임을 증명할 때, 그리고 EM 알고리즘·VAE의 ELBO를 유도할 때 Jensen's Inequality가 반드시 등장합니다.

볼록 함수(Convex Function)란?

두 점 사이의 함수 값보다 두 점을 잇는 직선이 항상 위에 있으면 볼록 함수(convex)입니다. 반대면 오목 함수(concave)입니다.

볼록 함수 (Convex) — ∪ 모양
  • \(f(x) = x^2\) (포물선)
  • \(f(x) = e^x\)
  • \(f(x) = -\log x\) (마이너스 로그)

정의: \(f(\lambda x + (1-\lambda)y) \leq \lambda f(x) + (1-\lambda)f(y)\)

오목 함수 (Concave) — ∩ 모양
  • \(f(x) = \log x\)
  • \(f(x) = \sqrt{x}\)
  • \(f(x) = -x^2\)

볼록 함수에 음수 부호를 붙이면 오목 함수

JENSEN'S INEQUALITY
\[ f\!\left(\mathbb{E}[X]\right) \leq \mathbb{E}\!\left[f(X)\right] \quad \text{(볼록 함수일 때)} \]
\(f\): 볼록 함수 (convex function)
\(\mathbb{E}[X]\): 확률 변수 X의 기댓값
\(\mathbb{E}[f(X)]\): f(X)의 기댓값
직관: 볼록 함수에서는 "평균의 함수값 ≤ 함수값의 평균"
오목 함수(예: log)에서는 부등호 반전: \(f(\mathbb{E}[X]) \geq \mathbb{E}[f(X)]\)

KL 비음수성 증명 — Jensen 적용

\(D_{KL}(P\|Q) \geq 0\)임을 Jensen's Inequality로 증명할 수 있습니다. \(\log\)는 오목 함수이므로 부등호가 반전됩니다.

KL ≥ 0 PROOF VIA JENSEN
\[ -D_{KL}(P\|Q) = \sum_x P(x) \log\frac{Q(x)}{P(x)} \leq \log\sum_x P(x)\frac{Q(x)}{P(x)} = \log 1 = 0 \]
핵심 단계: \(\log\)가 오목(concave)이므로 Jensen에 의해 \(\mathbb{E}[\log Z] \leq \log \mathbb{E}[Z]\)
\(\sum_x P(x) \cdot \frac{Q(x)}{P(x)} = \sum_x Q(x) = 1\)이므로 우변 = \(\log 1 = 0\)
따라서 \(-D_{KL}(P\|Q) \leq 0 \implies D_{KL}(P\|Q) \geq 0\)
등호 조건: \(P = Q\)일 때만 0

ELBO — VAE와 EM 알고리즘의 핵심

변분 추론(Variational Inference)에서 증거 하한(ELBO)을 유도할 때도 Jensen이 핵심입니다.

ELBO DERIVATION (Evidence Lower BOund)
\[ \log P(x) = \log \mathbb{E}_{z \sim q}\!\left[\frac{P(x,z)}{q(z)}\right] \geq \mathbb{E}_{z \sim q}\!\left[\log\frac{P(x,z)}{q(z)}\right] = \mathcal{L} \text{ (ELBO)} \]
\(P(x)\): 관측 데이터의 주변 우도 (계산 불가 — 잠재 변수 z를 적분해야 함)
\(q(z)\): 잠재 변수의 근사 분포 (우리가 최적화하는 분포)
Jensen(\(\log\)는 오목): \(\log \mathbb{E}[Y] \geq \mathbb{E}[\log Y]\) 적용
결과: \(\log P(x) \geq \mathcal{L}\) — ELBO를 최대화하면 진짜 로그우도를 간접적으로 최대화
\(\log P(x) - \mathcal{L} = D_{KL}(q(z)\|P(z \mid x)) \geq 0\) (항상 성립)
✅

논문 읽기 포인트: "by Jensen's inequality", "ELBO", "variational lower bound" — 이 표현들이 보이면 Jensen's Inequality가 적용된 것입니다. VAE(Kingma & Welling 2013), EM Algorithm 논문의 핵심 단계입니다.
