볼록 집합과 볼록 함수

"오목한 그릇"에 공을 굴리면 반드시 바닥에 도달한다. 볼록 함수의 최솟값은 항상 유일하고 보장됩니다.

🧒 초등학생 한 줄 설명

"오목한 그릇"에 공을 놓으면 어디서 놓아도 바닥으로 굴러간다 — 이것이 볼록 함수의 핵심입니다.

  • ① \(x^2\)은 볼록: 두 점을 직선으로 이으면 그 선이 항상 함수 위에 있다 → 공이 중앙 바닥으로 굴러감
  • ② \(\sin(x)\)은 비볼록: 여러 봉우리와 골짜기가 있어서 공이 어느 골짜기에 멈출지 모른다
  • ③ AI: Logistic loss(로지스틱 손실)는 볼록함수 → 경사하강법으로 반드시 최적해에 도달. 딥러닝은 비볼록 → 국소 최솟값이나 안장점에 걸릴 수 있다
🔗

Ch.17 연결: 앞 챕터에서 경사하강법이 "수렴한다"는 주장을 배웠습니다. 그 수렴 보장의 핵심 조건이 바로 볼록성(convexity)입니다. 볼록 함수에서만 국소 최솟값 = 전역 최솟값이 보장됩니다.

⬡볼록 집합 (Convex Set)

집합 \(C\)가 볼록(convex)이라는 것은 집합 안의 임의의 두 점을 연결하는 선분이 모두 집합 안에 들어온다는 뜻입니다. 직관적으로 "움푹 파인 곳 없이 꽉 찬 모양"이 볼록 집합입니다.

볼록 집합의 정의
\[ C \text{ 가 볼록} \iff \forall x, y \in C,\ \forall \lambda \in [0,1]:\ \lambda x + (1-\lambda) y \in C \]
\(C\): 볼록 여부를 판단할 집합
\(x, y \in C\): 집합 안의 임의의 두 점
\(\lambda \in [0,1]\): 두 점 사이를 보간하는 가중치 (0이면 \(y\), 1이면 \(x\))
\(\lambda x + (1-\lambda)y\): 두 점 \(x\)와 \(y\)를 잇는 선분 위의 점
선분 전체가 집합 안에 있어야 볼록 집합

볼록 집합 vs 비볼록 집합 예시

구분예시이유AI 연결
볼록 \(\mathbb{R}^n\) 전체실수 공간 전체모든 선분이 실수 공간 안에 존재파라미터 공간 \(\theta \in \mathbb{R}^n\)
볼록 공(Ball)\(\{x : \|x\| \leq r\}\)두 점 연결 선분이 항상 공 안에 있음L2 정규화의 feasible set

볼록 집합의 교집합은 볼록: \(C_1\)과 \(C_2\)가 둘 다 볼록이면 \(C_1 \cap C_2\)도 볼록입니다. ML에서 여러 제약 조건(L1, L2 노름 제약 등)의 교집합이 볼록임을 보장하는 핵심 성질입니다.

∪볼록 함수 (Convex Function)

함수 \(f\)가 볼록(convex)이라는 것은 함수의 그래프가 "아래로 볼록한 그릇" 모양이라는 것입니다. 임의의 두 점 \(x, y\)를 이은 현(chord)이 항상 함수 그래프 위에 있거나 같아야 합니다.

볼록 함수의 정의 (Jensen 부등식)
\[ f \text{ 가 볼록} \iff f\!\bigl(\lambda x + (1-\lambda)y\bigr) \leq \lambda f(x) + (1-\lambda)f(y),\quad \forall x,y,\ \forall \lambda \in [0,1] \]
\(f(\lambda x + (1-\lambda)y)\): 두 점 사이의 중간점에서의 함수값 (그래프 위의 점)
\(\lambda f(x) + (1-\lambda)f(y)\): 두 함수값의 가중 평균 (현 위의 점)
"함수값 ≤ 현" — 그래프가 현보다 아래에 있어야 볼록
등호는 \(f\)가 선형일 때 성립 (아핀 함수는 볼록이자 오목)

볼록 함수의 황금 성질

🔑 황금 성질 — 왜 볼록성이 중요한가

국소 최솟값 = 전역 최솟값: 볼록 함수에서는 경사하강법이 찾은 극솟값이 반드시 전역 최솟값입니다. 더 낮은 곳이 없습니다.

경사하강법 수렴 보장: 볼록 함수 + 적절한 학습률 → 경사하강법이 반드시 최솟값으로 수렴합니다.

Hessian 조건: 두 번 미분 가능한 함수라면, Hessian 행렬이 양반정치(positive semidefinite)인 것과 볼록성이 동치입니다.

볼록 함수의 2계 조건 (Hessian)
\[ f \text{ 가 볼록} \iff \nabla^2 f(x) \succeq 0 \quad \forall x \in \text{dom}(f) \]
\(\nabla^2 f(x)\): \(f\)의 Hessian 행렬 (2차 편미분 행렬)
\(\succeq 0\): 양반정치(positive semidefinite) — 모든 고유값 \(\geq 0\)
\(f(x) = x^2\): \(\nabla^2 f = 2 > 0\) → 볼록 ✅
\(f(x) = -x^2\): \(\nabla^2 f = -2 < 0\) → 오목 (convex 아님)
다변수: \(\mathbf{v}^\top \nabla^2 f(x)\, \mathbf{v} \geq 0\) for all \(\mathbf{v}\) 이면 볼록

JJensen 부등식과 AI 응용

Jensen 부등식은 볼록 함수의 정의를 기대값으로 확장한 것입니다. 볼록 함수에서는 함수의 기대값이 기대값의 함수보다 항상 크거나 같습니다.

Jensen 부등식
\[ f \text{ 가 볼록} \implies f\!\bigl(\mathbb{E}[X]\bigr) \leq \mathbb{E}\!\bigl[f(X)\bigr] \]
\(X\): 임의의 확률변수
\(\mathbb{E}[X]\): \(X\)의 기댓값 (평균)
\(f(\mathbb{E}[X])\): 평균값에서의 함수값 (그래프의 중간점)
\(\mathbb{E}[f(X)]\): 함수값의 평균 (현의 중간점)
볼록 → 함수값의 평균 ≥ 평균의 함수값 (현이 그래프 위에 있으므로)
\(f\)가 오목(concave)이면 부등호 방향 반전: \(f(\mathbb{E}[X]) \geq \mathbb{E}[f(X)]\)

KL 발산이 항상 0 이상인 이유 — Jensen으로 증명

정보이론에서 KL 발산 \(\mathrm{KL}(p \| q) \geq 0\)은 매우 중요한 성질입니다. 이를 Jensen 부등식으로 한 줄에 증명할 수 있습니다.

KL ≥ 0 증명 (Jensen 적용)
\[ \mathrm{KL}(p \| q) = -\mathbb{E}_p\!\left[\log\frac{q}{p}\right] \geq -\log\,\mathbb{E}_p\!\left[\frac{q}{p}\right] = -\log 1 = 0 \]
\(\mathrm{KL}(p \| q) = \mathbb{E}_p[\log(p/q)] = -\mathbb{E}_p[\log(q/p)]\): KL 발산 정의
\(\log\) 함수는 오목(concave) 함수 → Jensen 부등식 적용 시 부등호 반전
오목함수에서 Jensen: \(\mathbb{E}[\log Z] \leq \log \mathbb{E}[Z]\) (즉 \(-\mathbb{E}[\log Z] \geq -\log \mathbb{E}[Z]\))
\(\mathbb{E}_p[q/p] = \int p \cdot (q/p)\,dx = \int q\,dx = 1\): \(q\)가 확률분포이므로
따라서 \(\mathrm{KL}(p \| q) \geq -\log 1 = 0\) — Jensen 한 줄로 증명 완료
📄

논문 연결: VAE(Variational Autoencoder) 논문에서 ELBO(Evidence Lower BOund)가 Jensen 부등식으로 유도됩니다. \(\log p(x) \geq \mathbb{E}_q[\log p(x,z)] - \mathbb{E}_q[\log q(z)]\) — 이것이 VAE의 핵심 식입니다.

볼록 함수에서 Jensen
  • \(f(\mathbb{E}[X]) \leq \mathbb{E}[f(X)]\)
  • 예: \(f = e^x\) (지수함수는 볼록)
  • \(e^{\mathbb{E}[X]} \leq \mathbb{E}[e^X]\)
  • → moment generating function 하한
오목 함수에서 Jensen
  • \(f(\mathbb{E}[X]) \geq \mathbb{E}[f(X)]\)
  • 예: \(f = \log x\) (로그는 오목)
  • \(\log \mathbb{E}[X] \geq \mathbb{E}[\log X]\)
  • → KL ≥ 0 증명의 핵심
