연속성과 미분의 엄밀한 정의

Lipschitz 연속과 미분가능성 조건. 논문에서 "Gradient Lipschitz"가 왜 등장하는지 이해합니다.

🧒 초등학생 한 줄 설명

"미끄럼틀은 연속(펜 안 떼도 그릴 수 있음), 계단은 불연속(확 튀어 오름)"

  • x를 조금 바꿔도 f(x)가 조금만 바뀐다 = 연속 — 함수 그래프를 손으로 그릴 때 펜을 떼지 않아도 된다면 연속이다
  • Lipschitz 연속: 변화 속도에 상한이 있다 — 아무리 x를 빠르게 움직여도 f(x)의 변화 속도는 L배를 넘지 않는다
  • AI에서: 논문에 "f is L-smooth" = Gradient가 Lipschitz 연속 = 경사하강법 수렴 증명의 핵심 조건이다
🔗

Ch.15 연결: 지난 챕터의 ε-N 논증(수열 극한)과 이 챕터의 ε-δ 논증(함수 연속)은 같은 구조입니다. N 대신 δ(입력 범위), ε(출력 허용 오차)을 쓰는 것만 다릅니다.

〰연속의 ε-δ 정의

함수 \(f\)가 점 \(a\)에서 연속이라는 것은 "\(a\) 근방에서 입력을 조금만 바꾸면 출력도 조금만 바뀐다"는 직관을 수학적으로 엄밀히 표현한 것입니다. Ch.15의 ε-N 논증과 완전히 같은 구조입니다. \(\delta\)는 입력 범위, \(\varepsilon\)은 출력 허용 오차입니다.

함수 연속의 ε-δ 정의
\[ f \text{가 } a \text{에서 연속} \iff \forall\,\varepsilon > 0,\ \exists\,\delta > 0\ \text{s.t.}\ |x - a| < \delta \Rightarrow |f(x) - f(a)| < \varepsilon \]
\(\forall\,\varepsilon > 0\): 상대방이 아무리 작은 출력 오차를 요구해도
\(\exists\,\delta > 0\): 내가 그에 맞는 입력 허용 범위 \(\delta\)를 제시할 수 있다
\(|x - a| < \delta\): 입력 \(x\)가 \(a\)로부터 \(\delta\) 이내에 있으면
\(|f(x) - f(a)| < \varepsilon\): 출력 \(f(x)\)가 \(f(a)\)로부터 \(\varepsilon\) 이내에 있음을 보장한다
Ch.15의 ε-N과의 차이: 수열은 \(n > N\), 함수는 \(|x-a| < \delta\) — 구조는 동일하다
✅

극값 정리 (Extreme Value Theorem): 연속함수는 컴팩트 집합(닫힌 유계 집합) 위에서 반드시 최솟값과 최댓값이 존재한다. 이는 손실함수 \(\mathcal{L}\)의 최솟값이 존재함을 보장하는 수학적 근거다.

📐Lipschitz 연속

연속보다 강한 조건입니다. Lipschitz 연속 함수는 "기울기가 무한히 가팔라지지 않는" 함수입니다. 상수 \(L\)은 함수의 변화 속도에 대한 전역적 상한입니다. \(L\)이 작을수록 함수가 더 완만합니다.

Lipschitz 연속 조건
\[ |f(x) - f(y)| \leq L \cdot \|x - y\|, \quad \forall\, x, y \in X \]
\(L\): Lipschitz 상수 — 함수 변화 속도의 상한 (\(L \geq 0\))
\(\|x - y\|\): 입력 간의 거리 (1차원이면 \(|x-y|\), 고차원이면 유클리드 노름)
\(X\): 함수의 정의역 전체에 걸쳐 이 조건이 성립해야 한다
직관: 두 점의 입력 차이가 \(d\)이면 출력 차이는 최대 \(L \cdot d\) — 이 비율이 무한히 커지지 않는다

함수별 Lipschitz 상수 비교

함수Lipschitz 상수 \(L\)설명AI 활용
항등함수 \(f(x) = x\)\(L = 1\)기울기 = 1 (상수)Residual connection
\(f(x) = \sin x\)\(L = 1\)도함수 \(|\cos x| \leq 1\)Positional encoding
\(f(x) = x^2\) (전체 \(\mathbb{R}\))\(L = \infty\)기울기가 무한히 커짐무한 구간에선 불성립
ReLU \(f(x) = \max(0, x)\)\(L = 1\)기울기 0 또는 1활성화 함수 선택 이유
Sigmoid \(\sigma(x)\)\(L = 0.25\)최대 기울기 \(\sigma'(0) = 0.25\)Vanishing gradient 이유
📌

\(f\)가 미분가능하면 Lipschitz 상수 \(L = \sup_x |f'(x)|\)로 구할 수 있습니다. ReLU의 \(L=1\), Sigmoid의 \(L=0.25\)가 이렇게 나옵니다. 미분 불가능한 함수라도 Lipschitz 조건을 만족할 수 있습니다 (ReLU가 그 예).

∇L-smooth — 그라디언트 Lipschitz

AI 논문에서 가장 자주 등장하는 조건입니다. "f is L-smooth"는 f 자체가 아니라 gradient \(\nabla f\)가 Lipschitz 연속임을 의미합니다. 이 조건 아래에서 경사하강법의 수렴이 보장됩니다.

L-smooth (그라디언트 Lipschitz) 정의
\[ \|\nabla f(x) - \nabla f(y)\| \leq L \cdot \|x - y\|, \quad \forall\, x, y \]
\(\nabla f(x)\): 점 \(x\)에서의 gradient (벡터)
\(\|\nabla f(x) - \nabla f(y)\|\): 두 점에서의 gradient 차이의 크기
\(L\): smoothness 상수 — 클수록 곡률이 크다 (더 가파른 2차 함수 같은 모양)
직관: gradient가 급격히 바뀌지 않는다 → 2차 근사가 실제 함수를 잘 상한한다
L-smooth 동치 조건 — 2차 상한 부등식
\[ f(y) \leq f(x) + \nabla f(x)^\top (y - x) + \frac{L}{2} \|y - x\|^2 \]
위 조건은 L-smooth와 동치다 — 함수가 2차 함수(포물선)로 위에서 눌려 있는 모양
\(\nabla f(x)^\top (y - x)\): 점 \(x\)에서의 선형 근사
\(\frac{L}{2} \|y - x\|^2\): 2차 보정 항 — \(L\)이 클수록 더 넓은 포물선 상한
경사하강법 분석: 학습률 \(\alpha \leq 1/L\)로 설정하면 매 스텝 손실이 감소함이 보장된다
⚠️

Lipschitz ≠ L-smooth 혼동 주의: "f is L-Lipschitz"는 f 자체의 변화에 상한 (함수값 차이 ≤ L·거리), "f is L-smooth"는 gradient의 변화에 상한 (gradient 차이 ≤ L·거리)입니다. 논문에서 Assumption A1, A2로 각각 나눠 가정하는 경우가 많습니다.

🔑 논문에서 L-smooth가 쓰이는 이유

논문의 수렴 증명 흐름: (1) L-smooth 가정 → (2) 학습률 \(\alpha \leq 1/L\) 설정 → (3) 매 스텝 손실 감소량 하한 도출 → (4) T번 반복 후 수렴 보장

예: "Under Assumption A1 (L-smooth) and A2 (convex), with step size \(\alpha = 1/L\), GD satisfies \(\|\nabla f(x_T)\|^2 \leq 2L(f(x_0) - f^*)/T\)."

f'미분가능성과 연속성의 위계

미분가능성은 연속성보다 강한 조건입니다. 미분가능하면 연속이지만, 연속이라고 미분가능한 것은 아닙니다. ReLU가 대표적인 예입니다. 함수의 부드러움 정도를 \(C^k\) 클래스로 분류합니다.

미분의 정의 (극한으로)
\[ f'(a) = \lim_{h \to 0} \frac{f(a+h) - f(a)}{h} \]
\(h\): 입력의 미소 변화량 (\(h \to 0\) 의 방향은 양수, 음수 모두 같은 극한이어야 미분가능)
\(f'(a)\): 점 \(a\)에서의 도함수 — 기울기(순간 변화율)
Ch.15 극한 정의와 연결: \(\lim_{h \to 0}\)도 ε-δ 논증으로 엄밀히 정의된다
좌미분 = 우미분이어야 \(f'(a)\) 존재 — ReLU는 \(x=0\)에서 좌미분(-1)≠우미분(0)이므로 미분불가

함수 부드러움의 위계 (C⁰ → C∞)

클래스조건예시특징
\(C^0\) (연속)연속이지만 미분 불가ReLU그래프에 꺾임이 있음

  • x=0에서 꺾임 → 기울기 점프

  • GELU: \(C^\infty\), 논문 증명에 유리
  • Hessian 계산 가능 → 2차 최적화 가능

논문에서 "f is L-smooth (L-Lipschitz gradient)"의 정확한 의미는?

(a)   f 자체가 Lipschitz 연속이다 — \(|f(x)-f(y)| \leq L\|x-y\|\)

(c)   함수값이 L보다 작다 — \(f(x) \leq L\)
(d)   f가 L번 미분 가능하다 — \(f \in C^L\)
