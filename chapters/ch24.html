λ고유값과 고유벡터

정방행렬 \(A \in \mathbb{R}^{n \times n}\)에 의한 선형 변환은 대부분의 벡터를 회전시킵니다. 그런데 어떤 특별한 벡터 \(\mathbf{v}\)는 방향은 그대로이고 크기만 \(\lambda\)배 바뀝니다. 이 벡터를 고유벡터(eigenvector), 배율 \(\lambda\)를 고유값(eigenvalue)이라 합니다.

고유값·고유벡터 정의
\[ A\mathbf{v} = \lambda \mathbf{v}, \quad \mathbf{v} \neq \mathbf{0} \]
\(A\): \(n \times n\) 정방행렬
\(\mathbf{v}\): 고유벡터 — 변환 후에도 같은 방향을 유지하는 영벡터가 아닌 벡터
\(\lambda\): 고유값 — 고유벡터가 늘어나거나 줄어드는 배율 (음수 가능)
고유값은 \(\det(A - \lambda I) = 0\) (특성방정식)을 풀어서 구한다

대각화 — 행렬 거듭제곱을 쉽게 계산하는 열쇠

\(A\)가 \(n\)개의 선형 독립인 고유벡터를 가지면 대각화할 수 있습니다. 대각화는 반복 연산(정규화, 멱급수, 지수행렬)을 극적으로 단순화합니다.

대각화 (Eigendecomposition)
\[ A = V \Lambda V^{-1} \] \[ \Lambda = \operatorname{diag}(\lambda_1, \ldots, \lambda_n), \quad V = [\mathbf{v}_1 \mid \cdots \mid \mathbf{v}_n] \]
\(V\): 고유벡터를 열로 쌓은 행렬 — \(V\)의 \(i\)번째 열이 \(\mathbf{v}_i\)
\(\Lambda\): 고유값을 대각선에 놓은 대각행렬
\(V^{-1}\): \(V\)의 역행렬 — 고유벡터가 직교하면 \(V^{-1} = V^\top\)
핵심 응용: \(A^k = V \Lambda^k V^{-1}\) — 대각행렬의 거듭제곱은 각 원소를 \(k\)제곱하면 끝
💡

\(A^k = V\Lambda^k V^{-1}\)의 위력: Transformer의 반복 attention 합성, 마팅게일 이론의 전이행렬 분석 등 "행렬을 여러 번 곱하는" 상황마다 대각화가 등장한다. \(\lambda_i^k\)만 계산하면 되므로 \(O(n^3k)\)가 \(O(n^3 + nk)\)로 줄어든다.

고유값 찾기 — 특성다항식

단계수식의미
특성방정식 설정\(\det(A - \lambda I) = 0\)\(\lambda\)가 해가 되는 조건
고유값 \(\lambda_i\) 계산다항식 풀기최대 \(n\)개의 실수/복소수 해
고유벡터 \(\mathbf{v}_i\) 계산\((A - \lambda_i I)\mathbf{v} = \mathbf{0}\) 풀기영공간(null space) 구하기
대각화 조건 확인\(V\) 가역 여부고유벡터가 선형 독립이어야 함

S스펙트럼 정리 — 대칭행렬의 아름다운 구조

일반 행렬은 복소수 고유값이나 선형 종속 고유벡터를 가질 수 있습니다. 하지만 대칭행렬 \(A = A^\top\)는 훨씬 강력한 보장을 제공합니다. 바로 스펙트럼 정리(Spectral Theorem)입니다.

스펙트럼 정리 (Spectral Theorem)
\[ A = A^\top \implies A = Q \Lambda Q^\top \]
\(A = A^\top\): 대칭행렬 조건 — 공분산 행렬, Hessian 등이 여기에 해당
\(Q\): 직교행렬 (\(Q^{-1} = Q^\top\), \(QQ^\top = I\)) — 고유벡터가 서로 수직이고 단위벡터
\(\Lambda\): 실수 고유값으로 이루어진 대각행렬 — 복소수 고유값 없음이 보장됨
해석: \(A\)는 "회전(\(Q^\top\)) → 축별 스케일링(\(\Lambda\)) → 역회전(\(Q\))"으로 분해된다
일반 행렬
  • 고유값이 복소수일 수 있음
  • 고유벡터가 직교하지 않을 수 있음
  • \(V^{-1} \neq V^\top\) (계산 비쌈)
  • 대각화 불가능한 경우도 있음
대칭행렬 (스펙트럼 정리)
  • 고유값이 항상 실수
  • 고유벡터가 항상 직교
  • \(Q^{-1} = Q^\top\) (계산 빠름)
  • 항상 대각화 가능
📌

공분산 행렬에 스펙트럼 정리 적용: 데이터 행렬 \(X\)의 공분산 행렬 \(\Sigma = \frac{1}{n}X^\top X\)는 대칭·양반정치입니다. 스펙트럼 정리로 \(\Sigma = Q\Lambda Q^\top\)로 분해하면, \(Q\)의 열벡터가 주성분(PC) 방향이 됩니다 — 이것이 PCA의 수학적 본질입니다.

≽양반정치 행렬 — PSD 조건과 판별

대칭행렬 중에서도 특히 중요한 부류가 양반정치(Positive Semi-Definite, PSD) 행렬입니다. 공분산 행렬, Gram 행렬, Hessian 등 ML에 나오는 핵심 행렬 대부분이 여기에 속합니다.

양반정치 (PSD) 조건
\[ A \succeq 0 \iff \mathbf{x}^\top A \mathbf{x} \geq 0 \quad \forall \mathbf{x} \iff \text{모든 고유값 } \lambda_i \geq 0 \]
\(A \succeq 0\): 양반정치 기호 — "행렬 부등식"으로 읽음
\(\mathbf{x}^\top A \mathbf{x}\): 이차형식(quadratic form) — 스칼라값
세 조건은 동치: 어느 하나를 확인하면 나머지가 모두 성립
양정치(PD): 모든 고유값 \(\lambda_i > 0\) (등호 제외)

행렬 유형 분류

유형기호고유값 조건예시
양정치 (PD)\(A \succ 0\)모든 \(\lambda_i > 0\)강볼록 함수의 Hessian
양반정치 (PSD)\(A \succeq 0\)모든 \(\lambda_i \geq 0\)공분산 행렬, \(X^\top X\)
부정치 (Indefinite)—양수·음수 혼재비볼록 함수의 Hessian (안장점)
음정치 (ND)\(A \prec 0\)모든 \(\lambda_i < 0\)\(-\Sigma\) (음의 공분산)
⚠️

흔한 오해: "대각선 원소가 모두 양수이면 PSD"는 틀렸습니다. 예: \(\begin{pmatrix}1 & 10 \\ 10 & 1\end{pmatrix}\)은 대각선이 양수지만 고유값이 \(-9, 11\)이므로 부정치입니다. 반드시 모든 고유값 또는 이차형식 조건을 확인해야 합니다.

RRayleigh Quotient와 PCA

방향 벡터 \(\mathbf{x}\)가 행렬 \(A\)와 만날 때 "얼마나 증폭되는가"를 측정하는 값이 Rayleigh Quotient입니다. 이것을 최대화하면 자연스럽게 PCA의 첫 번째 주성분을 얻습니다.

Rayleigh Quotient 정의와 최적값
\[ R(A, \mathbf{x}) = \frac{\mathbf{x}^\top A \mathbf{x}}{\mathbf{x}^\top \mathbf{x}}, \qquad \lambda_{\min} \leq R(A, \mathbf{x}) \leq \lambda_{\max} \] \[ \max_{\mathbf{x} \neq \mathbf{0}} R(A, \mathbf{x}) = \lambda_{\max}, \quad \text{최대를 달성하는 } \mathbf{x} = \mathbf{v}_{\max} \]
\(R(A, \mathbf{x})\): Rayleigh Quotient — 벡터 크기 정규화 후 이차형식 값
\(\lambda_{\max}\): \(A\)의 최대 고유값
\(\mathbf{v}_{\max}\): 최대 고유값에 대응하는 고유벡터
PCA 연결: 공분산 행렬 \(\Sigma\)에 적용 → 분산이 최대인 방향 = 제1 주성분
🔑 PCA = 공분산 행렬의 고유값 분해

목표: 데이터의 분산이 가장 큰 방향(축)을 찾는다.

수학: 공분산 행렬 \(\Sigma \in \mathbb{R}^{d \times d}\)를 스펙트럼 분해 \(\Sigma = Q\Lambda Q^\top\)한다.

결과: \(Q\)의 열 = 주성분 방향 (서로 직교), \(\Lambda\)의 대각 = 각 방향의 분산

설명 분산 비율: \(k\)개 주성분이 설명하는 분산 = \(\displaystyle\frac{\lambda_1 + \cdots + \lambda_k}{\lambda_1 + \cdots + \lambda_d}\)
