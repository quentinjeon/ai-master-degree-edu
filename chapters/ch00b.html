Ch.1B · 기초 수학 브릿지 (2주차)

수열과 Σ·Π 표기

AI 논문 어디에나 등장하는 시그마(Σ)와 파이(Π) 기호를 자연스럽게 읽고 씁니다. 손실함수, 우도, 배치 평균 모두 Σ로 쓰입니다.

🧒 쉽게 이해하기

Σ는 "다 더해라"는 명령어, Π는 "다 곱해라"는 명령어다. 영수증 합계를 계산하는 것과 같다.

  • Σ: 마트에서 산 물건 10개의 가격을 모두 더한다 → \(\sum_{i=1}^{10} \text{가격}_i\). 학생 30명의 점수 평균 = \(\frac{1}{30}\sum_{i=1}^{30} x_i\).
  • 수열: 계단처럼 규칙이 있는 숫자 목록. 1, 2, 3, 4, … (등차수열), 1, 2, 4, 8, … (등비수열). 학습률이 점점 줄어드는 것도 등비수열!
  • AI 연결: 배치 손실 \(\frac{1}{n}\sum_{i=1}^{n}\mathcal{L}(f_\theta(x_i), y_i)\), 우도 \(\prod_{i=1}^{n} p(x_i|\theta)\) — 논문의 모든 평균과 곱이 Σ, Π로 표현된다.

Σ Σ (시그마) 표기 완전 해부

Σ는 그리스 문자로 "S(sum)"에 해당합니다. 반복되는 덧셈을 간결하게 쓰는 수학 약속입니다.

Σ 표기의 구조
\[ \sum_{i=1}^{n} f(i) = f(1) + f(2) + f(3) + \cdots + f(n) \]
\(i\): 인덱스 변수 (루프 변수처럼 1씩 증가)
\(i=1\): 시작값 (Σ 아래에 표기)
\(n\): 끝값 (Σ 위에 표기)
\(f(i)\): 각 단계에서 더할 항 — 인덱스에 따라 바뀌는 식

Σ 읽기 연습

수식풀어쓰면결과AI에서의 의미
\(\sum_{i=1}^{4} i\) \(1+2+3+4\) \(10\) 인덱스 합산
\(\sum_{i=1}^{n} x_i\) \(x_1+x_2+\cdots+x_n\) — n개 데이터의 합
\(\frac{1}{n}\sum_{i=1}^{n} x_i\) 합을 n으로 나눔 \(\bar{x}\) 표본평균 / 배치 평균
\(\sum_{i=1}^{n} (x_i - \bar{x})^2\) 편차 제곱의 합 — 표본분산의 분자
\(\sum_{k=1}^{K} y_k \log \hat{p}_k\) 클래스별 손실 합산 — Cross-Entropy Loss
✅

Σ 계산 팁: \(\sum_{i=1}^{n} c = nc\) (상수는 n번 더해짐), \(\sum_{i=1}^{n}(a_i + b_i) = \sum a_i + \sum b_i\) (분리 가능). 이 선형성이 배치 손실 분리에 핵심입니다.

📐 수열 — 규칙 있는 숫자 목록

수열은 순서가 있는 수의 나열입니다. 딥러닝에서 학습률 스케줄링, 반복 스텝, 레이어 깊이 등이 모두 수열의 개념을 사용합니다.

등차수열과 등비수열
\[ \text{등차: } a_n = a_1 + (n-1)d \qquad \text{등비: } a_n = a_1 \cdot r^{n-1} \]
등차수열 \(d\): 공차 (매번 더해지는 고정값). 예: 1, 3, 5, 7, … (d=2)
등비수열 \(r\): 공비 (매번 곱해지는 고정값). 예: 1, 0.5, 0.25, … (r=0.5)
AI 연결: 학습률 스케줄 \(\eta_t = \eta_0 \cdot \gamma^t\) 는 등비수열 (γ<1이면 감소)

등비수열의 수렴 — 경사하강법과의 연결

공비 \(|r| < 1\)이면 등비수열은 0으로 수렴합니다. 경사하강법에서 학습률 \(\eta\)가 적절하면 파라미터 업데이트 크기가 점점 줄어들며 수렴합니다.

등비급수 (무한 합)
\[ \sum_{k=0}^{\infty} r^k = \frac{1}{1-r} \quad (|r| < 1) \]
\(|r| < 1\): 수렴 조건 — 각 항이 점점 작아짐
예: \(1 + 0.5 + 0.25 + 0.125 + \cdots = \frac{1}{1-0.5} = 2\)
AI 연결: RNN의 그레디언트 소실 — 역전파 시 \(r < 1\)인 등비급수처럼 그레디언트가 0으로 사라짐

Π Π (파이) 표기 — 곱셈 버전의 Σ

Π는 반복되는 곱셈을 간결하게 쓰는 기호입니다. 확률론에서 독립 사건들의 결합 확률을 표현할 때 핵심적으로 쓰입니다.

Π 표기의 구조
\[ \prod_{i=1}^{n} f(i) = f(1) \times f(2) \times \cdots \times f(n) \]
구조는 Σ와 동일하지만 더하기 대신 곱하기
\(\prod_{i=1}^{n} p(x_i|\theta)\): n개의 독립 샘플 관측 확률의 곱 = likelihood
주의: 0~1 사이 값을 여러 번 곱하면 매우 빠르게 0에 가까워짐 → log로 변환 필요
🔑 Σ vs Π — 딥러닝 손실함수 연결

MLE: \(\hat{\theta} = \arg\max_\theta \prod_{i=1}^n p(x_i|\theta) = \arg\max_\theta \sum_{i=1}^n \log p(x_i|\theta)\)

Π → log → Σ: 곱을 합으로 변환. 이것이 로그우도를 쓰는 수학적 이유입니다. \(\ln(\prod a_i) = \sum \ln a_i\) 성질 덕분.

🔢 Σ·Π 계산기

\(\sum_{i=1}^{n} i^p\) 와 \(\prod_{i=1}^{n} r^i\) 를 직접 계산해보세요.

항 수 \(n\)
지수 \(p\)
공비 \(r\)
결과가 여기에 표시됩니다.
