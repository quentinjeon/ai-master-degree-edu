마팅게일과 확률 과정

"공정한 게임"의 수학. Doob's inequality, Optional Stopping Theorem. SGD 수렴 증명의 확률론적 기반.

🧒 초등학생 한 줄 설명

"카지노에서 아무리 전략을 써도 평균적으로 돈을 딸 수 없는 '공정한 게임'"이 마팅게일입니다. 미래 기댓값이 현재 값과 같다는 조건 — 이것이 SGD의 그라디언트 노이즈를 분석하는 수학적 언어입니다.

  • 마팅게일: 코인 게임에서 누적 수익 — \(\mathbb{E}[S_n \mid S_1,\ldots,S_{n-1}] = S_{n-1}\) (미래 기댓값 = 현재)
  • Doob's inequality: 마팅게일이 크게 뛸 확률의 상한 — 집중 부등식의 일반화
  • AI에서: SGD의 파라미터 업데이트 수열이 마팅게일류(martingale-like)이고, 이 성질로 수렴을 증명

🌊확률 과정 기초

확률 과정은 시간 인덱스를 가진 확률변수의 모임입니다. 각 시각 \(t\)마다 하나의 확률변수 \(X_t\)가 정의되며, 모든 시각의 집합이 바로 확률 과정입니다. 머신러닝에서 SGD 반복 수열 \(\theta_0, \theta_1, \theta_2, \ldots\)이 대표적인 확률 과정입니다.

STOCHASTIC PROCESS
\[ \{X_t\}_{t=0}^\infty,\quad X_t:\Omega\to\mathbb{R}^d \]
\(\{X_t\}\): 확률 과정 — 시간 \(t\)로 인덱싱된 확률변수들의 모임
\(X_t\): 각 시각 \(t\)에서 표본공간 \(\Omega\)에서 \(\mathbb{R}^d\)로 가는 확률변수
\(t = 0,1,2,\ldots\): 이산 시간 인덱스 (또는 \(t \in \mathbb{R}_+\)이면 연속 시간)
샘플 경로(sample path): 고정된 \(\omega \in \Omega\)에서 \(t \mapsto X_t(\omega)\)의 궤적
ML에서: \(\theta_0, \theta_1, \theta_2, \ldots\) (SGD 반복)이 확률 과정

중요한 확률 과정 유형

과정정의ML 연결
마코프 체인 \(P(X_t \mid X_0,\ldots,X_{t-1}) = P(X_t \mid X_{t-1})\) RL의 MDP
랜덤 워크 \(S_n = X_1 + \cdots + X_n\) SGD 경로
마팅게일 \(\mathbb{E}[X_n \mid X_1,\ldots,X_{n-1}] = X_{n-1}\) 공정한 게임, SGD
브라운 운동 연속 랜덤 워크 확률미분방정식

𝔼마팅게일 정의와 성질

마팅게일은 "공정한 게임"을 수학적으로 표현합니다. 핵심 조건은 현재까지의 모든 정보를 조건으로 할 때, 다음 단계의 기댓값이 현재 값과 같다는 것입니다. 이 성질 덕분에 SGD의 그라디언트 노이즈를 엄밀히 분석할 수 있습니다.

MARTINGALE DEFINITION
\[ \{M_t\}\text{ is a martingale w.r.t. }\{\mathcal{F}_t\} \iff \mathbb{E}[M_{t+1}\mid\mathcal{F}_t]=M_t\quad\forall t \]
\(\mathcal{F}_t\): \(t\) 시점까지의 정보를 담은 \(\sigma\)-대수 (filtration) — 과거와 현재의 모든 정보
\(\mathbb{E}[M_{t+1} \mid \mathcal{F}_t] = M_t\): 현재까지 알고 있는 모든 정보를 조건으로 할 때 미래 기댓값 = 현재
슈퍼마팅게일: \(\mathbb{E}[M_{t+1} \mid \mathcal{F}_t] \leq M_t\) (점점 감소하는 경향)
서브마팅게일: \(\mathbb{E}[M_{t+1} \mid \mathcal{F}_t] \geq M_t\) (점점 증가하는 경향)
SGD 적용: \(f(\theta_t)\)의 기댓값 수열이 슈퍼마팅게일 → 수렴 증명 가능
MARTINGALE DIFFERENCE SEQUENCE
\[ d_t = M_t - M_{t-1} \Rightarrow \mathbb{E}[d_t \mid \mathcal{F}_{t-1}] = 0 \quad\text{(마팅게일 차분)} \]
\(d_t\): 마팅게일 차분(martingale difference) — 각 단계의 변화량
\(\mathbb{E}[d_t \mid \mathcal{F}_{t-1}] = 0\): 각 단계 변화량의 조건부 기댓값 = 0
SGD: 미니배치 그라디언트 \(\tilde{\nabla}f(\theta)\)의 노이즈 \(\varepsilon_t = \tilde{\nabla}f(\theta_t) - \nabla f(\theta_t)\)가 마팅게일 차분
마팅게일 차분의 합: 집중 부등식을 적용할 수 있어 SGD 수렴 증명에 활용
💡

마팅게일과 독립성의 차이: 마팅게일 차분은 조건부 기댓값이 0인 것이지, 서로 독립일 필요는 없습니다. 독립보다 약한 조건이지만 집중 부등식을 적용하기에 충분합니다. 이것이 SGD 분석에서 마팅게일이 강력한 도구인 이유입니다.

DDoob의 부등식

마팅게일이 어느 시점에서나 임계값을 넘을 확률을 최대화하는 경계입니다. 마코프 부등식의 마팅게일 버전으로, 확률론적 알고리즘 분석의 핵심 도구입니다.

DOOB'S MAXIMAL INEQUALITY
\[ P\!\left(\max_{1\leq k\leq n}M_k\geq\lambda\right)\leq\frac{\mathbb{E}[M_n^+]}{\lambda} \quad\text{(비음 슈퍼마팅게일)} \]
\(M_n^+ = \max(M_n, 0)\): \(M_n\)의 양수 부분
\(\lambda > 0\): 임계값 (threshold)
\(\max_{1 \leq k \leq n} M_k\): \(n\) 시점까지 마팅게일의 최대값 — "어느 시점에서든 크게 뛸 확률"
마코프 부등식 \(P(X \geq \lambda) \leq \mathbb{E}[X]/\lambda\)의 마팅게일 버전
\(L^2\) 버전: \(P(\max_k |M_k| \geq \lambda) \leq \mathbb{E}[M_n^2] / \lambda^2\)
AZUMA-HOEFFDING INEQUALITY
\[ |d_t|\leq c_t\text{ a.s.}\Rightarrow P(M_n-M_0\geq t)\leq\exp\!\left(-\frac{t^2}{2\sum_{k=1}^n c_k^2}\right) \]
\(d_t = M_t - M_{t-1}\): 마팅게일 차분
\(|d_t| \leq c_t\): 각 단계 변화량이 \(c_t\)로 유계(bounded) — almost surely
\(\sum_{k=1}^n c_k^2\): 단계별 경계값의 제곱합 (분산 기여의 합)
McDiarmid 부등식(Ch.27)이 Azuma-Hoeffding의 특수 케이스
ML 적용: SGD에서 각 배치 그라디언트 변화량이 유계이면 이 경계 사용 → 수렴 속도 분석
📄

SGD 수렴 논문에서 "Let \(\varepsilon_t = \tilde{\nabla}f(\theta_t) - \nabla f(\theta_t)\) be a martingale difference sequence"라는 문장이 등장하면, 이는 그라디언트 노이즈에 Azuma-Hoeffding 부등식을 적용하기 위한 전제 조건입니다.

τOptional Stopping Theorem

공정한 게임(마팅게일)에서 언제 멈추는 전략을 써도 기대 수익은 처음과 같습니다. "더블링 전략"처럼 이기면 멈추는 전략이 기대수익을 높이지 못하는 이유가 이 정리에서 나옵니다.

OPTIONAL STOPPING THEOREM
\[ \tau\text{ 유계 stopping time},\ \{M_t\}\text{ 마팅게일} \Rightarrow \mathbb{E}[M_\tau]=\mathbb{E}[M_0] \]
\(\tau\): stopping time — 현재까지의 정보 \(\mathcal{F}_t\)만으로 정지 여부를 결정하는 랜덤 변수
\(\{\tau \leq t\} \in \mathcal{F}_t\): stopping time의 형식적 정의 — 미래 정보를 사용하지 않음
유계 조건: \(\tau \leq N\) a.s.이면 정리가 성립 (또는 uniformly integrable)
"더블링 전략" 실패 이유: 유한 자본에서 \(\tau\)가 유계가 아님 → 정리 적용 불가
ML 응용: 조기 종료(early stopping)가 평균적으로 bias를 도입하지 않는 조건 분석
⚠️

흔한 오해: "마팅게일이면 항상 E[M_τ] = E[M_0]"이라고 생각하기 쉽습니다. 그러나 τ가 유계가 아니거나 uniformly integrable 조건을 만족하지 않으면 이 등식이 깨집니다. 실제로 "더블링 전략"에서 τ는 유계가 아니고, 유한 자본이라는 외부 제약이 없으면 이론적으로 무한히 기다릴 수 있습니다.

마팅게일 핵심 성질 요약
  • E[M_t] = E[M_0]: 기댓값 보존
  • E[M_τ] = E[M_0]: OST (유계 τ)
  • Var(M_n) = Σ Var(dₜ): 분산 가법성
  • 차분 dₜ의 조건부 기댓값 = 0
SGD 수렴 증명에서의 활용
  • 그라디언트 노이즈 → 마팅게일 차분
  • Doob 부등식 → 일탈 확률 경계
  • Azuma-Hoeffding → 수렴 속도
  • 슈퍼마팅게일 → 손실 단조 감소

🔢 마팅게일 Random Walk — E[Sₙ] = S₀ 확인

대칭 랜덤 워크 \(S_n = \sum_{i=1}^n X_i,\ X_i \in \{+1,-1\}\text{ (각 1/2 확률)}\)을 시뮬레이션합니다. 마팅게일 성질: \(\mathbb{E}[S_n] = 0\)이지만 경로는 \(\sqrt{n}\) 정도 퍼집니다.
