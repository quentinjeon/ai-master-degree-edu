Ch.00E · 브릿지 (Level 0→1 · 수학 편)

수학 브릿지 — 기호와 표기법 읽는 법

논문에서 ∀, ∃, Σ, f: X→Y 같은 기호를 봐도 두렵지 않게 됩니다. Level 0에서 배운 직관을 수학 언어로 번역합니다.

🧒 초등학생 한 줄 설명

수학 기호는 새로운 내용이 아니라, "한국어로 알고 있는 것을 영어로 말하는 것처럼 — 번역이다."

  • "모든 학생의 점수가 0~100 사이" → ∀s ∈ Students, 0 ≤ score(s) ≤ 100
  • "평균 = 합 / 개수" → \(\bar{x} = \frac{1}{n}\sum_{i=1}^{n} x_i\)
  • "입력 넣으면 출력 나오는 기계" → f: X → Y, f(x) = wx + b
  • AI 연결: 논문의 모든 Theorem이 이 기호들로 쓰인다. 기호를 읽을 수 있으면 Theorem이 읽힌다.

∈ 집합 기호 — ∈, ⊂, ∪, ∩

집합은 "원소의 모임"입니다. 수학에서 "어떤 것이 어디에 속하는가"를 표현할 때 집합 기호를 씁니다.

기호읽는 법의미예시
∈"원소이다", "에 속한다"x가 집합 A의 원소x ∈ ℝ → "x는 실수"
∉"원소가 아니다"x가 집합 A에 없음-1 ∉ ℕ → "-1은 자연수 아님"
⊂"부분집합이다"A의 모든 원소가 B에도 있음ℕ ⊂ ℝ
∪"합집합"A 또는 B에 있는 원소A ∪ B
∩"교집합"A 이면서 B인 원소A ∩ B
ℕ자연수 집합{1, 2, 3, ...}n ∈ ℕ → "n은 자연수"
ℝ실수 집합모든 실수x ∈ ℝ → "x는 실수"
💡

논문 독해 팁: "Let x ∈ ℝⁿ be the input vector"를 보면 "x는 n차원 실수 벡터"라는 뜻이다. ℝⁿ = n개의 실수로 이루어진 집합. [1, 2, 3]은 ℝ³의 원소다.

∀ ∀ (모든)과 ∃ (어떤) — 논문 정리에서 가장 많이 나오는 기호

수학에서 "모든"과 "어떤"을 기호로 씁니다. 논문의 Theorem과 Assumption은 거의 모두 이 두 기호로 시작합니다.

한정사 기호 (Quantifiers)
\[ \forall x \in \mathbb{R},\ f(x) \geq 0 \quad\text{("모든 실수 } x \text{에 대해 } f(x) \text{는 0 이상이다")} \] \[ \exists\, x \in \mathbb{R}\ \text{s.t.}\ f(x) = 0 \quad\text{("0이 되는 } x \text{가 실수 중 하나 존재한다")} \]
\(\forall\) (for all): "모든 ~ 에 대해"
\(\exists\) (there exists): "~ 이 존재한다"
s.t. (such that): "~ 를 만족하는"
논문 예: "∀x ∈ 𝒳, ‖f(x)‖ ≤ L" → "모든 입력에 대해 f(x)의 크기가 L 이하"
Level 0 표현

"모든 학생의 점수는 0 이상이다"

"어떤 학생은 100점을 받았다"

Level 1 수학 표기

\(\forall s \in \text{Students},\ \text{score}(s) \geq 0\)

\(\exists s \in \text{Students}\ \text{s.t.}\ \text{score}(s) = 100\)

✅

극한의 ε-N 정의: \(\forall\varepsilon > 0,\ \exists N\ \text{s.t.}\ n > N \Rightarrow |a_n - L| < \varepsilon\) — 암호처럼 보이지만 번역하면 "아무리 작은 ε을 줘도 N이 존재해서 그 이후 모든 항이 L에서 ε 이내에 있다"이다. 기호만 알면 읽힌다.

f f: X → Y — 함수를 수학으로 정의하는 법

Level 0에서 def predict(x): return w*x + b로 함수를 만들었습니다. 수학에서는 이것을 f: ℝ → ℝ, f(x) = wx + b로 씁니다. 같은 개념입니다.

Python수학 표기읽는 법
def f(x):\(f: X \to Y\)f는 X에서 Y로 가는 함수
return w*x + b\(f(x) = wx + b\)f(x)의 값은 wx + b
f(3)\(f(3) = w \cdot 3 + b\)x=3일 때 f의 값
max(scores)\(\arg\max f(x)\)f를 최대로 하는 x
min(loss)\(\arg\min_\theta L(\theta)\)손실 L을 최소로 하는 θ
함수의 수학적 정의
\[ f: \mathbb{R}^n \to \mathbb{R} \quad\text{where}\quad f(\mathbf{x}) = \mathbf{w}^\top \mathbf{x} + b \]
\(f: \mathbb{R}^n \to \mathbb{R}\): n개의 숫자를 입력받아 숫자 하나를 출력
\(\mathbf{w}^\top \mathbf{x}\): 가중치 벡터와 입력 벡터의 내적 (weighted sum)
\(b\): 편향 (기본값 오프셋)
이것이 선형회귀, 로지스틱 회귀 등 모든 선형 모델의 공통 형태

Σ Σ 표기 — 반복 합산을 한 줄로

Level 0에서 sum(scores)를 썼습니다. 수학에서는 Σ(시그마)를 씁니다. 새로운 개념이 아니라 같은 연산을 기호로 쓴 것입니다.

Σ 표기법과 Python 대응
\[ \sum_{i=1}^{n} x_i = x_1 + x_2 + \cdots + x_n \quad\Leftrightarrow\quad \texttt{sum(x)} \] \[ \bar{x} = \frac{1}{n}\sum_{i=1}^{n} x_i \quad\Leftrightarrow\quad \texttt{np.mean(x)} \]
\(i=1\) (아래): 시작값 — i가 1부터 증가
\(n\) (위): 끝값 — n까지 증가
\(x_i\): i번째 원소를 더함
\(\bar{x}\) (x bar): 평균 — 전체 합을 개수로 나눈 값
🔑 Σ 읽는 법 4단계
1

아래 확인: 시작값 (i=1 → 1부터)

2

위 확인: 끝값 (n → n까지)

3

뒤 확인: 더할 것 (xᵢ → i번째 원소)

4

전체: i를 1~n까지 바꿔가며 xᵢ를 모두 더함

🔢 함수 시각화 — w와 b를 바꿔보세요

슬라이더로 가중치 \(w\)와 편향 \(b\)를 조절하면 \(f(x) = wx + b\) 함수값이 실시간으로 바뀝니다.

가중치 \(w\)
편향 \(b\)
결과가 여기에 표시됩니다.
