VC 차원과 Rademacher 복잡도

무한 가설 클래스의 복잡도를 측정합니다. 신경망의 일반화 경계, "VC dimension d → n = O(d/ε²)"의 수학.

🧒 초등학생 한 줄 설명

선형 분류기로 평면 위의 점들을 얼마나 다양하게 나눌 수 있는가? — "3개 점은 어떻게 놓아도 가능하지만 4개 점은 불가능한 배치가 있다. 이 '최대 분류 가능 점 수'가 VC 차원이다."

  • 1D 임계값 분류기: VC 차원 = 1 (점 1개는 분류 가능, 2개는 불가)
  • 2D 선형 분류기: VC 차원 = 3 (3점은 가능, 4점은 XOR 패턴 불가)
  • AI에서: 신경망의 VC 차원 = O(W log W) (W = 파라미터 수) → 일반화 경계가 이를 통해 유도되지만, 딥러닝의 실제 성능은 이론보다 훨씬 좋다

VShattering과 VC 차원

가설 클래스 \(\mathcal{H}\)가 점 집합 \(C\)를 분쇄(shatter)한다는 것은 \(C\)의 모든 가능한 이진 라벨 할당을 \(\mathcal{H}\) 안의 어떤 \(h\)가 완벽히 분류할 수 있다는 뜻입니다. VC 차원은 분쇄 가능한 최대 집합의 크기입니다.

VC DIMENSION DEFINITION
\[ \mathrm{VCdim}(\mathcal{H}) = \max\{|C|:\mathcal{H}\text{ shatters }C\} \]
\(C\): 점들의 집합
"\(\mathcal{H}\)가 \(C\)를 분쇄": \(\forall(y_1,\ldots,y_{|C|}) \in \{0,1\}^{|C|},\ \exists h \in \mathcal{H}\ \text{s.t.}\ h(x_i)=y_i\ \text{for all}\ i\)
\(\mathrm{VCdim}=d\)이면 크기 \(d\)인 집합은 분쇄 가능, 크기 \(d+1\)인 모든 집합은 분쇄 불가

주요 가설 클래스의 VC 차원

가설 클래스차원VC 차원
1D 임계값 \(h(x)=\mathbf{1}[x>\theta]\)11
2D 선형 분류기23
\(d\)차원 선형 분류기\(d\)\(d+1\)
신경망 (\(W\) 파라미터)—\(O(W \log W)\)
결정트리 (깊이 \(d\))—\(O(2^d)\)
📌

분쇄의 핵심: \(\mathcal{H}\)가 크기 \(d+1\)인 점 집합을 분쇄하지 못한다는 것이 포인트입니다. "어떤 배치를 잡아도 분쇄 불가"이어야 VC 차원이 \(d\)로 확정됩니다. 하나라도 분쇄 가능한 배치가 존재하면 VC 차원은 적어도 \(d+1\)입니다.

∗통계적 학습의 기본 정리

VC 차원이 유한하면 PAC 학습 가능하고, 무한하면 PAC 학습 불가능합니다. 이것이 통계적 학습 이론의 핵심 결과입니다. VC 경계는 훈련 오차와 검증 오차의 차이를 표본 크기와 VC 차원으로 한정합니다.

VC GENERALIZATION BOUND
\[ P\!\left[L_D(h_S)-L_S(h_S)\leq\sqrt{\frac{2d\ln(en/d)+2\ln(4/\delta)}{n}}\right]\geq 1-\delta \]
\(d = \mathrm{VCdim}(\mathcal{H})\): VC 차원
우변이 \(\varepsilon\)이 되도록 풀면 표본 복잡도: \(n = O\!\left(d/\varepsilon^2 + \log(1/\delta)/\varepsilon^2\right)\)
상수 \(e \approx 2.718\)은 자연상수 (Sauer's Lemma에서 유래)
핵심: \(n\)이 충분하면 훈련-검증 오차 차이가 \(\varepsilon\) 이하
VC SAMPLE COMPLEXITY
\[ m_\mathcal{H}(\varepsilon,\delta) = O\!\left(\frac{d+\log(1/\delta)}{\varepsilon^2}\right) \]
\(d\) = VC 차원, \(\varepsilon\) = 정확도, \(\delta\) = 신뢰도
\(d\)차원 선형 분류기: \(m_\mathcal{H} = O(d/\varepsilon^2)\)
신경망(\(W\) 파라미터): \(m_\mathcal{H} = O(W \log W/\varepsilon^2)\)
✅ VC 경계의 장점
  • 분포와 무관한 최악의 경우 경계
  • PAC 학습 가능성의 필요충분 조건
  • 표본 복잡도를 VC 차원 하나로 요약
  • 임의 가설 클래스에 적용 가능
⚠️ VC 경계의 한계
  • 신경망에서 너무 느슨 (실제보다 훨씬 큰 \(n\) 요구)
  • 분포 정보를 전혀 활용하지 않음
  • 딥러닝의 "더블 디센트" 현상 설명 불가
  • 회귀·다중 클래스에는 직접 적용 불가

RRademacher 복잡도

VC 차원은 이진 분류에만 적용되고 분포와 무관합니다. Rademacher 복잡도는 특정 분포와 표본에 의존하는 더 정밀한 복잡도 척도입니다. "무작위 노이즈 라벨과 얼마나 상관관계를 가질 수 있는가"를 측정합니다.

EMPIRICAL RADEMACHER COMPLEXITY
\[ \hat{\mathcal{R}}_S(\mathcal{H}) = \mathbb{E}_{\boldsymbol{\sigma}}\!\left[\sup_{h\in\mathcal{H}}\frac{1}{n}\sum_{i=1}^n\sigma_i h(x_i)\right] \]
\(\sigma_i \in \{-1,+1\}\): i.i.d. Rademacher 변수(균등 분포) — 무작위 노이즈 라벨
\(\sup_{h\in\mathcal{H}}\): \(\mathcal{H}\) 안의 \(h\) 중 노이즈와 가장 잘 상관되는 것을 선택
값이 클수록 \(\mathcal{H}\)가 더 복잡 (노이즈에도 맞출 수 있음을 의미)
Rademacher 복잡도 = 0 → 복잡도 없음 (가설이 하나뿐인 경우 등)
RADEMACHER GENERALIZATION BOUND
\[ P\!\left[L_D(h)\leq L_S(h)+2\mathcal{R}_n(\mathcal{H})+\sqrt{\frac{\ln(2/\delta)}{2n}}\right]\geq 1-\delta \]
\(\mathcal{R}_n(\mathcal{H})\): 표본 크기 \(n\)에서의 Rademacher 복잡도
VC 경계보다 분포에 따라 더 타이트할 수 있음 (분포 정보를 활용하기 때문)
딥러닝에서: 실제로 Rademacher 복잡도가 낮아서 이론이 예측하는 것보다 잘 일반화됨
\(2\mathcal{R}_n(\mathcal{H})\): 복잡도 페널티 항 / \(\sqrt{\ln(2/\delta)/2n}\): 확률 신뢰도 항
💡

VC vs Rademacher 비교: VC 차원은 분포와 무관한 "최악의 경우" 경계이고, Rademacher 복잡도는 실제 데이터 분포와 표본에 의존하는 "평균적 경우" 경계입니다. 실제 딥러닝 모델의 일반화를 분석할 때 Rademacher가 더 유용할 때가 많습니다.

SSauer's Lemma — 무한 ℋ의 수학적 도구

유한 \(\mathcal{H}\)에 대한 일반화 경계는 Union Bound로 유도됩니다. 무한 \(\mathcal{H}\)(예: 선형 분류기, 신경망)에 대해서는 "성장 함수(growth function)"를 통해 유효한 "패턴 수"를 유한으로 제한합니다. Sauer's Lemma가 이 핵심 도구입니다.

🔑 성장 함수 (Growth Function)

성장 함수 \(\Pi_\mathcal{H}(n)\)는 \(n\)개 점에서 \(\mathcal{H}\)가 만들 수 있는 서로 다른 이진 패턴(이분)의 최대 개수입니다.

\[ \Pi_\mathcal{H}(n) = \max_{x_1,\ldots,x_n \in \mathcal{X}} |\{(h(x_1),\ldots,h(x_n)) : h \in \mathcal{H}\}| \]
SAUER'S LEMMA
\[ \Pi_\mathcal{H}(n) \leq \sum_{i=0}^{d}\binom{n}{i} \leq \left(\frac{en}{d}\right)^d \]
\(\Pi_\mathcal{H}(n)\): 성장 함수 = \(n\)개 점에서 \(\mathcal{H}\)가 만들 수 있는 서로 다른 이진 패턴 수
\(d = \mathrm{VCdim}(\mathcal{H})\)
\(n \leq d\)이면 최대 \(2^n\), \(n > d\)이면 다항식(\(n^d\))으로 증가
이 다항식 증가가 일반화 경계 유도의 핵심 — Union Bound를 다항식 개의 가설로 제한
⚠️

흔한 오해: 신경망의 VC 차원이 크면 항상 일반화가 나쁘다고 생각하기 쉽습니다. 그러나 딥러닝의 "double descent" 현상이 보여주듯, VC 차원이 표본 수보다 훨씬 커도 실제 일반화 성능이 좋을 수 있습니다. VC 이론은 최악의 경우 경계이며, 실제 딥러닝에서는 더 정밀한 분석이 필요합니다.
