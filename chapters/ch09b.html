Ch.9B · ML 연결 (브릿지)

모델 평가지표
— Precision, Recall, F1, ROC·AUC

분류 모델이 "얼마나 잘했는지"를 정확히 측정하는 지표들을 익히면, ML 논문의 실험 결과 표를 정확히 읽을 수 있습니다.

🧒 초등학생 한 줄 설명

모델이 맞춘 것과 틀린 것을 "맞춤 비율"과 "빠뜨린 비율"로 나눠서 점수를 매깁니다.

  • 스팸 메일 걸러내기 — 진짜 스팸을 못 잡으면 낮은 Recall, 일반 메일을 스팸으로 잘못 잡으면 낮은 Precision
  • 암 진단 — 암인데 정상 판정(놓침) vs 정상인데 암 판정(불필요한 치료) — 어느 게 더 나쁜지 상황마다 다름
  • AI 논문에서 표의 숫자가 F1=0.92 → Precision과 Recall의 조화 평균이 0.92라는 뜻

Accuracy의 함정: 암 환자 비율이 1%인 데이터에서 "모두 정상"이라 예측하면 Accuracy=99%가 나옵니다. 이런 불균형 데이터에서는 Accuracy가 의미 없으므로 Precision/Recall/F1을 사용해야 합니다.

🎯Precision · Recall · F1

PRECISION — 정밀도
\[ \text{Precision} = \frac{TP}{TP + FP} \]
\(TP\): True Positive — 실제 양성을 양성으로 맞게 예측한 수
\(FP\): False Positive — 실제 음성을 양성으로 잘못 예측한 수
의미: "양성이라 예측한 것 중 실제로 양성인 비율" — 모델의 정확성
예) 스팸 필터: 스팸으로 분류한 메일 중 진짜 스팸의 비율
RECALL (SENSITIVITY) — 재현율
\[ \text{Recall} = \frac{TP}{TP + FN} \]
\(FN\): False Negative — 실제 양성을 음성으로 놓친 수
의미: "실제 양성 중 모델이 잡아낸 비율" — 모델의 커버리지
예) 암 진단: 실제 암 환자 중 양성 판정 받은 비율 — 높을수록 암을 놓치지 않음
민감도(Sensitivity)와 동의어; 반대 개념은 특이도(Specificity) = TN/(TN+FP)
F1 SCORE — 조화 평균
\[ F_1 = \frac{2 \cdot \text{Precision} \cdot \text{Recall}}{\text{Precision} + \text{Recall}} = \frac{2 \cdot TP}{2 \cdot TP + FP + FN} \]
의미: Precision과 Recall의 조화 평균 — 둘 다 높아야 F1도 높아짐
산술 평균과 달리 조화 평균은 낮은 쪽 값에 더 많은 가중치를 줌
예) Precision=1.0, Recall=0.1이면 F1 = 2×0.1/(1+0.1) ≈ 0.182 (낮음)

Precision-Recall Trade-off

분류 임계값(threshold)을 높이면 Precision ↑, Recall ↓. 낮추면 Recall ↑, Precision ↓. 이 상충 관계를 PR Curve로 시각화합니다.

Precision 우선 상황
  • 스팸 필터 (일반 메일 손실 방지)
  • 법원 판결 (무고한 사람 처벌 방지)
  • 추천 시스템 (관련 없는 추천 최소화)
Recall 우선 상황
  • 암 진단 (암 놓치면 치명적)
  • 사기 탐지 (사기 거래 놓치면 손실)
  • 바이러스 검사 (감염자 놓치면 확산)

📈ROC 곡선과 AUC

ROC(Receiver Operating Characteristic) 곡선은 임계값을 0→1로 변화시키면서 TPR(Recall)과 FPR의 관계를 그린 것입니다.

ROC CURVE — TPR vs FPR
\[ \text{TPR} = \frac{TP}{TP+FN} \quad (\text{Recall}), \qquad \text{FPR} = \frac{FP}{FP+TN} \]
\(\text{TPR}\): True Positive Rate — 양성 중 맞게 양성 예측한 비율
\(\text{FPR}\): False Positive Rate — 음성 중 잘못 양성 예측한 비율
완벽한 모델: ROC 곡선이 (0,0) → (0,1) → (1,1) 꺾임 (좌상단 코너에 붙음)
랜덤 모델: ROC 곡선이 대각선 (y=x)
AUC — Area Under the ROC Curve
\[ \text{AUC} = \int_0^1 \text{TPR}(\text{FPR}^{-1}(t)) \, dt \in [0, 1] \]
의미: ROC 곡선 아래 면적 — 클수록 모델 성능이 좋음
AUC = 1.0: 완벽한 분류
AUC = 0.5: 랜덤 분류 (동전 던지기 수준)
AUC < 0.5: 의도적으로 틀리는 모델 (레이블 반전하면 좋아짐)
해석: "임의로 뽑은 양성 샘플의 예측 점수가 음성 샘플보다 높을 확률"
