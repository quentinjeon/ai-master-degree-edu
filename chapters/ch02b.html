적분 기초

연속형 확률분포를 이해하려면 적분이 필수입니다. "확률 = 넓이"라는 직관이 PDF, 기대값, 정규화 상수를 하나로 연결합니다.

🧒 쉽게 이해하기

적분은 그래프 아래의 넓이를 구하는 것이다. 속도계를 보면서 얼마나 이동했는지 계산하는 것과 같다.

  • 속도 × 시간 = 거리: 시속 60km로 2시간을 달리면 120km. 속도가 계속 바뀐다면? → 그래프 아래 넓이로 구한다.
  • 연속 합산: Σ가 "이산적인 덧셈"이라면, 적분 ∫는 "연속적인 무한히 작은 조각들의 합"이다.
  • AI 연결: 연속형 확률분포 PDF에서 특정 구간의 확률 = 그 구간의 넓이. \(\int_{-\infty}^{\infty} f(x)dx = 1\) — 전체 확률은 항상 1이어야 한다.

∫ 정적분 — 넓이의 수학

정적분 \(\int_a^b f(x)\,dx\)는 함수 \(f(x)\)의 그래프와 x축 사이의 넓이를 \(a\)부터 \(b\)까지 구한 값입니다. 미분이 "순간 변화율"이라면, 적분은 "변화의 누적량"입니다.

정적분의 정의 (리만 합)
\[ \int_a^b f(x)\,dx = \lim_{n\to\infty} \sum_{i=1}^{n} f(x_i^*)\,\Delta x \quad \text{where } \Delta x = \frac{b-a}{n} \]
\([a,b]\)를 \(n\)개 작은 구간으로 나눔 → 각 구간을 직사각형으로 근사 → 무한히 세밀하게 나누면 정확한 넓이
\(\Delta x\): 각 직사각형의 너비 (n→∞이면 0에 수렴)
Σ와 ∫의 관계: ∫는 무한히 많은 무한히 작은 값들의 Σ

미적분학의 기본정리 — 미분과 적분의 연결

미분과 적분은 서로 역연산입니다. 이 관계를 이용하면 복잡한 극한 계산 없이 적분을 구할 수 있습니다.

미적분학의 기본정리
\[ \int_a^b f(x)\,dx = F(b) - F(a) \quad \text{where } F'(x) = f(x) \]
\(F(x)\): \(f(x)\)의 역도함수 (부정적분). \(F'(x) = f(x)\)를 만족하는 함수
의미: 넓이(적분)를 구하려면 역도함수(F)를 찾아 두 끝점에서 뺀다
예: \(\int_0^2 x\,dx = \left[\frac{x^2}{2}\right]_0^2 = \frac{4}{2} - 0 = 2\)

자주 쓰이는 적분 공식

함수 \(f(x)\)역도함수 \(F(x)\)AI 활용 상황
\(x^n\)\(\dfrac{x^{n+1}}{n+1} + C\)다항식 레이어 분석
\(e^x\)\(e^x + C\)정규분포 정규화 상수

📊 확률분포와 적분 — "확률 = 넓이"

연속형 확률변수의 확률은 반드시 적분으로 계산됩니다. 이것이 PDF(확률밀도함수)가 존재하는 이유입니다.

PDF의 핵심 성질
\[ P(a \le X \le b) = \int_a^b f(x)\,dx \qquad \int_{-\infty}^{\infty} f(x)\,dx = 1 \]
\(f(x)\): 확률밀도함수(PDF) — 특정 값의 "확률 밀도"를 나타냄
\(P(a \le X \le b)\): a와 b 사이의 확률 = PDF 그래프 아래 넓이
\(\int_{-\infty}^{\infty} f(x)dx = 1\): 전체 확률의 합은 반드시 1 (정규화 조건)
주의: \(f(x)\)는 확률이 아니라 밀도 — \(f(x) > 1\)도 가능
💡

정규분포와 적분: \(\mathcal{N}(x|\mu,\sigma^2)\)의 전체 넓이가 1임은 가우시안 적분 \(\int_{-\infty}^{\infty} e^{-x^2}dx = \sqrt{\pi}\)으로 증명됩니다. 딥러닝 논문에서 "normalized"라는 표현은 이 적분이 1이 되도록 했다는 뜻입니다.

⚖️ 연속형 기대값 — 적분으로 계산하는 평균

이산형 기대값 \(\mathbb{E}[X] = \sum_x x \cdot P(X=x)\)의 연속형 버전은 Σ가 ∫로 바뀝니다.

연속형 기대값과 분산
\[ \mathbb{E}[X] = \int_{-\infty}^{\infty} x \cdot f(x)\,dx \qquad \text{Var}(X) = \int_{-\infty}^{\infty} (x - \mathbb{E}[X])^2 \cdot f(x)\,dx \]
\(x \cdot f(x)\): 값 × 그 값의 밀도 — 가중 평균의 연속형
\(\mathbb{E}[g(X)] = \int g(x) f(x)\,dx\): 임의의 함수 g에 대한 기대값 (LOTUS 공식)
AI 연결: 손실함수의 기대값 \(\mathbb{E}_{(x,y)\sim p}[\mathcal{L}(f_\theta(x), y)]\) — 진짜 손실은 데이터 분포 전체에 대한 적분
🔑 정규분포의 기대값·분산 — 직접 계산

\(\mathcal{N}(x|\mu, \sigma^2)\)에 위 공식을 적용하면:

\[ \mathbb{E}[X] = \int_{-\infty}^{\infty} x \cdot \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(x-\mu)^2}{2\sigma^2}} dx = \mu \]

결과: 정규분포의 기대값 = \(\mu\) (분포의 중심). 이 적분이 가우시안 적분 \(\int x e^{-ax^2}dx\)의 계산을 통해 증명됩니다.

⚠️

흔한 혼동: PDF의 \(f(x)\) 값 자체는 확률이 아닙니다. \(P(X = x) = 0\) (연속형). 확률은 반드시 구간 적분으로만 얻을 수 있습니다: \(P(a \le X \le b) = \int_a^b f(x)dx\).

🔢 정규분포 구간 확률 계산기

\(\mathcal{N}(\mu=0, \sigma^2=1)\)에서 \([a, b]\) 구간의 확률 \(P(a \le X \le b)\)를 수치 적분으로 계산합니다.
