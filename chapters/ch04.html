Ch.4 · 확률 & 통계 (3~4주차)

확률론 기초

불확실성을 수학적으로 표현하는 언어. AI 모델의 예측은 본질적으로 확률입니다.

🧒 쉽게 이해하기 — 초등학생도 아는 확률

확률은 '어떤 일이 일어날 가능성을 0~1 사이 숫자로 나타낸 것'이다. 1이면 무조건, 0이면 절대 안 일어난다.

  • 🪙 동전 앞면이 나올 확률: 50% = 0.5
  • 🌧️ 내일 비가 올 확률: 30% = 0.3
  • 🐱 AI가 사진을 보고 판단: 고양이 0.9, 강아지 0.1 — AI는 확률로 생각한다!

🎲확률의 3가지 기본 공리 (Kolmogorov)

모든 확률 이론의 기초가 되는 3가지 공리입니다. 이 공리들로부터 나머지 모든 확률 법칙이 유도됩니다.

Kolmogorov 공리
\[ \text{1. } P(A) \ge 0 \qquad \text{2. } P(\Omega) = 1 \qquad \text{3. } P(A \cup B) = P(A) + P(B) \quad \text{(A, B 배반 시)} \]
Ω: 표본공간 (모든 결과의 집합)
공리 3: 배반(disjoint) 사건의 합집합 확률은 더한 값과 같다

핵심 용어

용어설명예시
표본공간 Ω발생 가능한 모든 결과동전: {앞, 뒤}
사건 A표본공간의 부분집합앞면만: {앞}
P(A)사건 A의 확률P({앞}) = 0.5
여사건 AᶜA가 아닌 사건뒷면: P(Aᶜ) = 1 − P(A)

|조건부확률 (Conditional Probability)

"B가 일어났음을 알게 됐을 때, A가 일어날 확률". 새 정보가 생기면 표본공간이 Ω에서 B로 줄어든다.

🌦️

비가 올 확률이 30%였는데, 먹구름을 봤다. 그러면 비 올 확률이 80%로 올라간다. 새 정보가 생기면 확률이 바뀐다!

조건부확률 공식
\[ P(A \mid B) = \frac{P(A \cap B)}{P(B)}, \quad P(B) > 0 \]
"B가 일어났음을 알게 됐을 때, A가 일어날 확률"
표본공간이 Ω에서 B로 줄어드는 것과 같다
🤖

AI에서 \(P(y \mid x)\)는 "입력 x가 주어졌을 때 y가 나올 확률" — 모든 분류기·회귀기의 학습 목표

🔄베이즈 정리 (Bayes' Theorem)

"처음 생각(prior)" + "새 정보(likelihood)" = "업데이트된 생각(posterior)"

🏥

병 검사 예시: 병이 드묾(prior 낮음) → 검사 양성(새 정보) → 하지만 검사기도 틀릴 수 있음! → posterior 계산 필요

베이즈 정리
\[ \underbrace{P(\theta \mid \mathcal{D})}_{\text{posterior}} = \frac{ \overbrace{P(\mathcal{D} \mid \theta)}^{\text{likelihood}} \cdot \overbrace{P(\theta)}^{\text{prior}} }{ \underbrace{P(\mathcal{D})}_{\text{evidence}} } \]
posterior: 데이터를 본 후 믿음  /  likelihood: θ일 때 데이터 관측 확률  /  prior: 데이터 보기 전 믿음  /  evidence: 정규화 상수, 보통 ∝ 를 써서 생략

4가지 요소

용어의미비유AI 예시
Prior P(θ)데이터 보기 전 믿음병원 가기 전 내 생각파라미터 초기 분포
Likelihood P(D|θ)θ일 때 D가 나올 확률이 동전으로 9번 앞면 가능성데이터 생성 확률
Posterior P(θ|D)D를 보고 업데이트된 믿음검사 결과 본 후 내 생각학습 후 파라미터 분포
Evidence P(D)정규화 상수기준점보통 ∝ 사용

단계별 베이즈 업데이트

1

Prior 설정 — 데이터 없이 파라미터에 대한 초기 믿음 설정

2

Likelihood 계산 — 현재 파라미터로 이 데이터가 나올 확률 계산

3

Posterior 업데이트 — 베이즈 정리로 믿음을 업데이트

4

반복 — 새 데이터가 올 때마다 posterior가 새로운 prior가 됨

⊥독립성과 조건부 독립 — 나이브 베이즈의 핵심

두 사건이 독립(Independent)이라는 것은 한 사건의 결과가 다른 사건의 확률에 영향을 주지 않는다는 뜻입니다. 이 개념은 나이브 베이즈, 그래픽 모델, 인과 추론 등 AI 논문에서 핵심적으로 등장합니다.

INDEPENDENCE — 독립 사건의 정의
\[ A \perp B \iff P(A \cap B) = P(A) \cdot P(B) \]
\(A \perp B\): "A와 B는 독립" 표기 (논문 표준 기호)
동치 조건: \(P(A \mid B) = P(A)\) — B를 알아도 A의 확률이 바뀌지 않음
예) 동전 앞면과 주사위 6면: 완전히 독립 → \(P(\text{앞면} \cap 6) = 0.5 \times \frac{1}{6}\)
반례) 비 오는 날과 우산 판매량: 독립이 아님 (연관됨)

조건부 독립 (Conditional Independence)

조건부 독립은 어떤 제3의 변수 C가 주어졌을 때 A와 B가 독립이 된다는 개념입니다. 나이브 베이즈 분류기와 베이지안 네트워크의 기반 가정입니다.

CONDITIONAL INDEPENDENCE
\[ A \perp B \mid C \iff P(A \cap B \mid C) = P(A \mid C) \cdot P(B \mid C) \]
\(A \perp B \mid C\): "C가 주어졌을 때 A와 B는 조건부 독립"
직관: C라는 공통 원인을 알면 A·B 사이의 연관성이 설명됨
예) 아이스크림 판매량(A)과 익사 사고(B)는 연관돼 보이지만, 기온(C)을 알면 조건부 독립
나이브 베이즈: \(P(x_1, x_2, \ldots, x_n \mid y) = \prod_i P(x_i \mid y)\) — y가 주어지면 모든 특성이 조건부 독립 가정

주변화 (Marginalization) — 복잡한 결합 확률 다루기

결합 확률 분포에서 특정 변수를 합산(적분)하여 제거하면 나머지 변수의 주변 분포를 얻습니다. 논문에서 \(P(x) = \sum_z P(x, z)\) 형태로 자주 등장합니다.

MARGINALIZATION — 주변화 공식
\[ P(A) = \sum_{b} P(A, B=b) = \sum_{b} P(A \mid B=b) P(B=b) \]
이산: \(P(A) = \sum_b P(A,B=b)\) — B에 대해 합산
연속: \(p(x) = \int p(x, z) \, dz\) — z에 대해 적분
응용: 잠재 변수 모델(VAE, GMM)에서 잠재 변수 \(z\)를 주변화하여 \(P(x)\) 계산
주의: 잠재 변수가 많으면 계산이 불가능해져 변분 추론(VI)이나 MCMC가 필요함
✅

논문 읽기 포인트: \(A \perp\!\!\!\perp B\), \(A \perp\!\!\!\perp B \mid C\) 기호를 보면 독립성 가정입니다. 나이브 베이즈·Hidden Markov Model·VAE 논문의 핵심 가정이니 꼭 기억하세요.
