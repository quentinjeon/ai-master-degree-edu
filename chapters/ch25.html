ΣSVD 정의 — 모든 행렬의 분해

특이값 분해(Singular Value Decomposition, SVD)는 임의의 \(m \times n\) 실수 행렬 \(A\)를 세 행렬의 곱으로 분해하는 방법입니다. 정방 행렬이 아니어도, 역행렬이 없어도 항상 존재합니다.

🔑 핵심 개념 — SVD의 기하학적 의미

임의의 선형변환 \(A\)는 다음 3단계로 해석할 수 있습니다: 회전(V^T) → 스케일(Σ) → 회전(U). 특이값 \(\sigma_i\)는 각 축 방향의 "늘림" 정도입니다.

이 분해는 항상 존재하며, 특이값을 내림차순으로 정렬하면 상위 \(k\)개 성분이 행렬의 핵심 정보를 담습니다.

SVD — 특이값 분해
\[ A = U \Sigma V^\top \]
\(A \in \mathbb{R}^{m \times n}\): 분해 대상 행렬 (임의의 크기)
\(U \in \mathbb{R}^{m \times m}\): 좌 특이벡터 행렬 — 직교행렬 (\(UU^\top = I_m\))
\(\Sigma \in \mathbb{R}^{m \times n}\): 대각 행렬 — 대각 원소 \(\sigma_1 \geq \sigma_2 \geq \cdots \geq \sigma_r \geq 0\)이 특이값
\(V \in \mathbb{R}^{n \times n}\): 우 특이벡터 행렬 — 직교행렬 (\(VV^\top = I_n\))
기하학: \(V^\top\)로 회전 → \(\Sigma\)로 스케일 → \(U\)로 회전
특이값 계산 — 고유값과의 관계
\[ A^\top A = V \Sigma^\top \Sigma V^\top = V \mathrm{diag}(\sigma_1^2, \ldots, \sigma_r^2) V^\top \quad \Rightarrow \quad \sigma_i = \sqrt{\lambda_i(A^\top A)} \]
\(A^\top A \in \mathbb{R}^{n \times n}\): 양반정치 대칭행렬 — 고유값이 항상 \(\geq 0\)
\(\lambda_i(A^\top A)\): \(A^\top A\)의 \(i\)번째 고유값 (내림차순)
\(\sigma_i\): \(i\)번째 특이값 = \(A^\top A\)의 고유값의 제곱근
\(V\)의 열벡터: \(A^\top A\)의 고유벡터 (우 특이벡터)
\(U\)의 열벡터: \(AV / \sigma_i\) (좌 특이벡터)

SVD 구성 요소 요약

구성 요소크기성질의미
\(U\) (좌 특이벡터)\(m \times m\)직교행렬출력 공간의 새 좌표계
\(\Sigma\) (특이값 대각)\(m \times n\)대각, \(\sigma_i \geq 0\)각 방향의 "중요도"
\(V^\top\) (우 특이벡터)\(n \times n\)직교행렬입력 공간의 새 좌표계

kEckart-Young 정리 — 최적 저랭크 근사

SVD의 핵심 응용은 저랭크 근사(low-rank approximation)입니다. 상위 \(k\)개의 특이값과 특이벡터만 사용하면 "가능한 모든 랭크-\(k\) 행렬 중 가장 좋은 근사"를 얻습니다. 이것이 Eckart-Young 정리입니다.

Eckart-Young 정리 — 최적 저랭크 근사
\[ A_k = \sum_{i=1}^{k} \sigma_i \mathbf{u}_i \mathbf{v}_i^\top = U_k \Sigma_k V_k^\top \quad = \quad \underset{\mathrm{rank}(B) \leq k}{\arg\min}\; \|A - B\|_F \]
\(A_k\): 상위 \(k\)개 특이 삼중쌍(singular triplet)으로 구성한 랭크-\(k\) 근사 행렬
\(\sigma_i \mathbf{u}_i \mathbf{v}_i^\top\): \(i\)번째 랭크-1 성분 (각각 독립 기여)
\(\|A - B\|_F\): Frobenius 노름 — 행렬 원소별 제곱합의 제곱근
오차: \(\|A - A_k\|_F^2 = \sigma_{k+1}^2 + \sigma_{k+2}^2 + \cdots + \sigma_r^2\)
이론 보장: 같은 랭크 제약에서 SVD 상위 \(k\)개 성분이 항상 최적 — PCA·이미지 압축·LoRA의 근거
✅

PCA와의 관계: 데이터 행렬 \(X\)를 SVD로 분해하면 주성분분석(PCA)과 정확히 동일합니다. \(V\)의 열벡터가 주성분 방향이고, \(\sigma_i^2\)이 분산을 설명합니다. Eckart-Young 정리가 PCA의 수학적 보장을 제공합니다.

📐

설명 분산 비율 계산: 상위 \(k\)개 성분으로 설명하는 분산 비율은 \(\displaystyle\frac{\sum_{i=1}^{k} \sigma_i^2}{\sum_{i=1}^{r} \sigma_i^2} \times 100\%\)로 계산합니다. 이 값이 90%를 넘으면 통상적으로 충분한 근사로 봅니다.

🎬응용: 행렬 완성 — 추천 시스템

실제 추천 시스템에서 유저-아이템 평점 행렬 \(M\)은 대부분이 비어 있습니다 (유저가 모든 영화를 보지 않으므로). SVD 기반 행렬 완성(matrix completion)은 관측된 평점으로부터 미관측 평점을 예측합니다.

행렬 완성 목적함수
\[ \min_{U, V} \sum_{(i,j) \in \Omega} \bigl(M_{ij} - \mathbf{u}_i^\top \mathbf{v}_j\bigr)^2 + \lambda \bigl(\|U\|_F^2 + \|V\|_F^2\bigr) \]
\(M_{ij}\): 유저 \(i\)가 아이템 \(j\)에 매긴 실제 평점
\(\Omega\): 관측된 (유저, 아이템) 쌍의 집합
\(\mathbf{u}_i \in \mathbb{R}^k\): 유저 \(i\)의 잠재 요인 벡터
\(\mathbf{v}_j \in \mathbb{R}^k\): 아이템 \(j\)의 잠재 요인 벡터
\(\lambda\): 정규화 계수 — 과적합 방지
예측: \(\hat{M}_{ij} = \mathbf{u}_i^\top \mathbf{v}_j\) (내적으로 미관측 평점 예측)
전통 SVD 방식
  • 완전 행렬에만 적용 가능
  • 누락값 처리 필요 (평균 대체 등)
  • Eckart-Young 이론 보장 있음
  • 계산량: \(O(\min(m,n)^2 \cdot \max(m,n))\)
행렬 인수분해 방식
  • 부분 관측 행렬에 직접 적용
  • 경사하강법으로 최적화
  • Netflix Prize에서 검증된 방식
  • 확장성 우수, 대규모 데이터 처리 가능

🤖LoRA — 저랭크 적응으로 LLM 파인튜닝

LoRA(Low-Rank Adaptation)는 사전 학습된 거대 언어 모델을 파인튜닝할 때, 전체 가중치 행렬을 업데이트하는 대신 가중치 변화량 \(\Delta W\)를 저랭크 행렬 두 개의 곱으로 근사합니다. Eckart-Young 정리가 이론적 근거를 제공합니다.

LoRA — 저랭크 가중치 적응
\[ W' = W_0 + \Delta W = W_0 + B A, \quad B \in \mathbb{R}^{d \times r},\; A \in \mathbb{R}^{r \times d},\; r \ll d \]
\(W_0 \in \mathbb{R}^{d \times d}\): 사전 학습된 원본 가중치 — 학습 중 고정(frozen)
\(\Delta W = BA\): 학습 가능한 가중치 변화량 — 랭크 \(r\)의 저랭크 행렬
\(B \in \mathbb{R}^{d \times r}\): 다운 프로젝션 — 0으로 초기화
\(A \in \mathbb{R}^{r \times d}\): 업 프로젝션 — 가우시안 랜덤 초기화
파라미터 수 비교: 원본 \(d^2\) → LoRA \(2dr\) (\(r=4, d=4096\)이면 99.8% 감소)
이론 근거: 파인튜닝 시 \(\Delta W\)의 내재 랭크가 낮다는 Eckart-Young 정리 활용
⚠️

흔한 오해: LoRA가 SVD를 직접 계산하는 것은 아닙니다. LoRA는 "가중치 변화가 저랭크 구조를 가질 것"이라는 가정 하에 \(B, A\)를 경사하강법으로 학습합니다. Eckart-Young 정리는 이 가정의 이론적 정당성을 제공합니다.
