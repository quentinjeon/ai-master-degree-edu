딥러닝 일반화 미스터리 — Double Descent·NTK

VC 이론이 예측하지 못한 딥러닝의 일반화. Double Descent 현상, 암묵적 정규화, NTK. 최신 연구의 핵심 질문.

🧒 초등학생 한 줄 설명

고전 이론은 "모델이 클수록 과적합"이라 예측하지만, GPT-4는 수천억 파라미터를 가지면서도 일반화가 잘 된다 — 이것이 딥러닝의 가장 큰 수수께끼입니다.

  • 전통 이론: 파라미터 수 > 데이터 수이면 완전 암기(overfitting) → 일반화 실패
  • 실제: 매우 큰 신경망은 오히려 더 잘 일반화된다 (Double Descent의 두 번째 하강)
  • AI에서: 이 현상을 설명하는 최신 이론 — 암묵적 정규화, NTK, benign overfitting — 이 세 개가 최신 ML 이론 논문의 핵심

⚖고전 Bias-Variance Tradeoff의 한계

고전적인 편향-분산 트레이드오프는 "어딘가 최적 복잡도가 있다"고 예측합니다. 모델이 너무 단순하면 편향이 높아 과소적합, 너무 복잡하면 분산이 높아 과적합이 발생한다는 고전적 U자 곡선입니다. 하지만 딥러닝의 과파라미터화 영역에서는 이 이론이 틀립니다.

BIAS-VARIANCE DECOMPOSITION
\[ \mathbb{E}\!\left[(f(x)-\hat{f}(x))^2\right] = \underbrace{(f(x)-\bar{f}(x))^2}_{\text{Bias}^2} + \underbrace{\mathbb{E}\!\left[(\hat{f}(x)-\bar{f}(x))^2\right]}_{\text{Variance}} + \sigma^2 \]
\(f(x)\): 진짜 함수 (알 수 없는 실제 목표)
\(\hat{f}(x)\): 학습된 모델의 예측값
\(\bar{f}(x) = \mathbb{E}[\hat{f}(x)]\): 기대 예측값 (여러 훈련 집합 평균)
\(\sigma^2\): 데이터 노이즈 — 어떤 모델로도 줄일 수 없는 비환원 오차
고전 이론: 복잡도↑ → Bias↓, Variance↑ → test error가 U자 형태
딥러닝: 과파라미터화 영역에서 Variance가 다시 감소 → Double Descent 현상 발생
⚠️

고전 이론의 예측 실패: 고전 통계학에서 파라미터 수 \(p\)가 데이터 수 \(n\)을 초과하면 과적합이 발생한다고 배웁니다. 그러나 실제 딥러닝 모델(ResNet, GPT 등)은 \(p \gg n\) 환경에서도 뛰어난 일반화를 보여줍니다. 이것이 현대 ML 이론의 핵심 연구 주제입니다.

〽Double Descent 현상

2019년 Belkin 등이 체계적으로 관찰한 Double Descent 현상: 모델 복잡도(파라미터 수)가 증가할 때, test error가 고전적 U자 곡선 이후 한 번 더 상승했다가 다시 감소합니다. 마치 두 번의 하강이 나타나는 이 현상은 딥러닝 일반화의 핵심 수수께끼입니다.

INTERPOLATION THRESHOLD
\[ \text{보간 임계값 (interpolation threshold):}\quad n_{\text{crit}} \approx n \cdot d_{\text{eff}} \]
\(n_{\text{crit}}\): 파라미터 수가 데이터 수와 같아지는 임계 지점 — 여기서 test error가 발산(spike)
\(n\): 훈련 데이터 수
\(d_{\text{eff}}\): 유효 차원 (effective dimension)
임계값 이전: 고전적 bias-variance tradeoff 영역 — 모델이 클수록 test error 증가
임계값 이후: "현대 영역" — 모델이 클수록 test error가 오히려 감소
\(n_{\text{crit}}\) 부근에서 최악의 일반화 성능이 관찰됨
🔑 Double Descent의 세 가지 형태

Double Descent는 단일 현상이 아닌 여러 맥락에서 관찰됩니다:

  • 모델 크기 Double Descent: 파라미터 수 증가 → test error 두 번 감소 (고전 영역 → spike → 현대 영역)
  • 에포크 Double Descent: 훈련 시간이 길어질수록 test error가 처음 오르다가 다시 감소
  • 표본 크기 Double Descent: 데이터가 늘어날 때도 유사한 비단조 패턴 존재
  • 공통 원인: 과파라미터화 모델이 "최소 놈(minimum norm)" 해를 선택 → 암묵적 정규화 효과

∇암묵적 정규화

과파라미터화 모델에는 훈련 오차 0을 달성하는 해가 무한히 많습니다. 어떤 해를 선택하느냐가 일반화 성능을 결정합니다. SGD는 명시적 정규화 없이도 최소 Frobenius 놈을 가진 해를 선택하는 경향이 있습니다 — 이것이 바로 암묵적 정규화(implicit regularization)입니다.

MINIMUM NORM SOLUTION
\[ \min_W \|W\|_F^2 \quad\text{s.t.}\quad XW=Y \quad\Rightarrow\quad \hat{W} = X^\dagger Y = X^\top(XX^\top)^{-1}Y \]
\(X^\dagger\): Moore-Penrose pseudoinverse — 최소 놈 해를 구해주는 연산자
\(\|W\|_F^2\): Frobenius 놈 제곱 — 행렬 원소 제곱합
\(XW = Y\): 훈련 데이터를 완전히 보간(interpolation)하는 조건
SGD는 제로 초기화에서 시작할 때 이 최소 놈 해로 수렴 (선형 모델에서 증명됨)
최소 놈 해 ≈ 암묵적 L2 정규화 ≈ Ridge regression과 유사한 효과
비선형 딥러닝에서도 유사한 편향이 실험적으로 관찰됨
📄

NeurIPS 2022 Best Paper: "Why neural networks find flat minima" — SGD는 loss landscape에서 더 "평평한(flat)" 최솟값을 선택하는 경향이 있고, 이것이 더 좋은 일반화로 이어진다는 이론. Sharp minima보다 flat minima에서 학습된 모델이 분포 변화에 더 강인합니다.

KNeural Tangent Kernel (NTK)

2018년 Jacot 등이 제안한 NTK(Neural Tangent Kernel): 신경망이 매우 넓어질수록(width→∞) 학습 역학이 커널 방법과 동치가 됩니다. 이를 통해 무한 너비 신경망의 학습을 선형 방정식으로 분석할 수 있게 되었습니다.

NEURAL TANGENT KERNEL
\[ K_\theta(x,x') = \nabla_\theta f(x;\theta)^\top \nabla_\theta f(x';\theta) \quad\Rightarrow\quad \frac{d\theta}{dt} = -\eta\, K_\theta(X,X)^{-1}(f(X;\theta)-y) \]
\(K_\theta(x,x')\): NTK — 두 데이터 포인트의 그라디언트 내적으로 정의되는 커널 함수
\(\nabla_\theta f(x;\theta)\): 파라미터 \(\theta\)에 대한 네트워크 출력의 그라디언트
\(\frac{d\theta}{dt}\): 연속 시간 경사하강법 (gradient flow)에서 파라미터 변화
무한 너비 극한에서 \(K_\theta \approx K_\infty\) — 상수, 학습 중 변하지 않음
이 경우 학습 역학이 선형화되어 수렴 속도와 일반화를 분석적으로 계산 가능
한계: 유한 너비(실제 딥러닝)에서 NTK는 학습 중 변함 — "feature learning" 설명 어려움

이론 핵심 주장 한계
