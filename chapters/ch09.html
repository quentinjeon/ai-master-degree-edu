Ch.9 · ML 연결 (11주차)

AI가 확률로 배우는 법

P(y|x)를 모델링하는 방법. Logistic Regression으로 확률론이 AI 학습에 연결되는 원리를 배웁니다.

🧒 초등학생도 이해하는 한 줄 비유

"AI는 '이 답이 맞을 가능성이 얼마나 큰가?'를 확률로 배우는 기계다." P(y=1|x)처럼 0~1 사이 숫자로 답을 낸다.

  • 📧 이메일이 스팸일 확률: P(스팸|이메일 내용)
  • 🐱 사진이 고양이일 확률: P(고양이|이미지)
  • 🤖 AI에서: Logistic Regression = P(y=1|x) = σ(wᵀx + b)

📚지도학습 (Supervised Learning)의 수학적 구조

지도학습은 입력 x에서 출력 y를 예측하는 함수 P(y|x)를 데이터로부터 학습하는 과정입니다.

구분 출력 타입 분포 손실함수
회귀 (Regression) 연속값 ŷ ∈ ℝ Gaussian MSE (L2 loss)
이진 분류 (Binary) 이진값 y ∈ {0, 1} Bernoulli Binary Cross-Entropy
다중 분류 (Multi) 범주형 y ∈ {0,...,K} Categorical Cross-Entropy

σ시그모이드 함수 (Sigmoid)

임의의 실수를 (0, 1) 사이의 확률로 변환하는 함수입니다.

시그모이드 함수
\[ \sigma(z) = \frac{1}{1 + e^{-z}} \in (0, 1) \qquad \sigma'(z) = \sigma(z)(1 - \sigma(z)) \]
z = 선형 변환값(= wᵀx + b)  |  출력: 0~1 사이의 확률값
미분: 자기 자신으로 표현됨 → 역전파 계산 편리

📈로지스틱 회귀 (Logistic Regression)

Logistic Regression — P(y=1|x) 모델링
\[ P(y=1 \mid \mathbf{x}; \mathbf{w}, b) = \sigma(\mathbf{w}^\top \mathbf{x} + b) = \frac{1}{1 + e^{-(\mathbf{w}^\top \mathbf{x} + b)}} \]
w = 가중치 벡터 (학습 대상)  |  b = 편향 (bias)  |  σ(·) = sigmoid — 확률로 변환
Binary Cross-Entropy Loss (BCE) = 음의 로그우도
\[ \mathcal{L}_{BCE} = -\frac{1}{n}\sum_{i=1}^{n}\left[ y_i \log \hat{p}_i + (1-y_i)\log(1-\hat{p}_i) \right] \]
yi ∈ {0, 1} = 정답 레이블  |  p̂i = σ(wᵀxi + b) = 모델의 예측 확률
y=1일 때: −log(p̂) — 확신할수록(p̂→1) 손실 작음
y=0일 때: −log(1−p̂) — 0에 확신할수록 손실 작음
🔑 BCE 최소화 = MLE

BCE를 최소화하는 것은 Bernoulli 분포에 대한 MLE와 정확히 동치. 딥러닝 분류 학습의 수학적 근거
