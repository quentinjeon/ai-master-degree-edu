선형회귀와 최소제곱법

고등학교 "회귀직선"이 바로 머신러닝의 첫 번째 모델입니다. 손실함수를 최소화하는 것이 학습임을 수식으로 이해합니다.

🧒 쉽게 이해하기

흩어진 점들에 가장 잘 맞는 직선을 긋는 것이 선형회귀다. 오차가 가장 작은 직선을 찾는다.

  • 키로 몸무게 예측: 키가 클수록 몸무게가 늘어나는 경향 → 점들을 직선으로 표현하면 키(x)만 알아도 몸무게(y)를 예측할 수 있다.
  • 잔차(오차): 실제값과 직선(예측값)의 차이. 이 오차들의 제곱합이 가장 작은 직선이 정답이다.
  • AI 연결: 선형회귀 \(\hat{y} = \mathbf{w}^\top\mathbf{x} + b\)는 모든 딥러닝 레이어의 기본 단위다. 뉴럴 네트워크 = 선형회귀를 여러 층으로 쌓고 활성화 함수로 비선형성 추가.

📈 선형회귀 모델

선형회귀는 입력 \(x\)와 출력 \(y\) 사이에 선형 관계를 가정하는 가장 단순한 예측 모델입니다. 고등학교에서 배운 "회귀직선 \(y = ax + b\)"이 바로 이것입니다.

단순 선형회귀 모델
\[ \hat{y} = wx + b \]
\(\hat{y}\): 예측값 (hat = 추정량을 의미)
\(w\): 가중치(weight) = 기울기. 고교 교과서의 \(a\)에 해당
\(b\): 편향(bias) = y절편. 고교 교과서의 \(b\)에 해당
목표: 주어진 데이터에서 최적의 \(w\)와 \(b\)를 찾는다
고등학교 표기

\(y = ax + b\)

  • \(a\): 기울기 (상관계수·공분산으로 계산)
  • \(b\): y절편
  • 최소제곱법으로 a, b 결정
머신러닝 표기

\(\hat{y} = w^\top x + b\)

  • \(w\): 가중치 벡터 (다차원 입력)
  • \(b\): 편향 스칼라
  • 손실함수 최소화로 w, b 결정

📉 MSE 손실함수 — "얼마나 틀렸는가"

좋은 직선이란 예측값과 실제값의 차이(잔차)가 작은 직선입니다. 이 차이를 수학적으로 표현한 것이 MSE(평균제곱오차) 손실함수입니다.

Mean Squared Error (MSE)
\[ \mathcal{L}(w, b) = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2 = \frac{1}{n}\sum_{i=1}^{n}(y_i - wx_i - b)^2 \]
\(y_i\): i번째 실제값, \(\hat{y}_i = wx_i + b\): i번째 예측값
\(y_i - \hat{y}_i\): 잔차(residual). 부호 상관없이 크면 나쁨 → 제곱으로 항상 양수
\(\frac{1}{n}\): n개 샘플 평균 (데이터 수에 상관없이 비교 가능하게)
목표: \(\mathcal{L}(w,b)\)을 최소화하는 \(w^*\), \(b^*\)를 찾아라
✅

왜 제곱인가? 잔차 \((y_i - \hat{y}_i)\)의 합은 양수·음수가 상쇄되어 0이 될 수 있습니다. 절댓값 대신 제곱을 쓰면 미분이 가능해져서 경사하강법을 적용할 수 있습니다.

🔢 최소제곱법 — 해석적 해

MSE를 \(w\)와 \(b\)로 편미분하여 0으로 놓으면 최적해를 수식으로 직접 구할 수 있습니다. 이것이 고등학교에서 배운 "최소제곱법 공식"의 수학적 유도입니다.

단순 선형회귀의 최소제곱 해
\[ w^* = \frac{\sum_{i=1}^{n}(x_i - \bar{x})(y_i - \bar{y})}{\sum_{i=1}^{n}(x_i - \bar{x})^2} = \frac{\text{Cov}(X,Y)}{\text{Var}(X)} \qquad b^* = \bar{y} - w^*\bar{x} \]
\(\bar{x}, \bar{y}\): 입력·출력의 표본평균
분자 \(\sum(x_i-\bar{x})(y_i-\bar{y})\): 공분산의 분자 — x와 y가 함께 변하는 정도
분모 \(\sum(x_i-\bar{x})^2\): 분산의 분자 — x의 퍼짐 정도
\(w^* = \text{Cov}(X,Y)/\text{Var}(X)\): 상관계수 × 표준편차 비율과 같음 (고교 공식과 동일!)

행렬 표기 — 다변량으로 확장

입력이 여러 개인 다중 선형회귀는 행렬로 간결하게 표현됩니다. Ch.3(선형대수)의 내용이 여기서 활용됩니다.

다중 선형회귀의 최소제곱 해 (OLS)
\[ \hat{\mathbf{w}} = (\mathbf{X}^\top \mathbf{X})^{-1} \mathbf{X}^\top \mathbf{y} \]
\(\mathbf{X} \in \mathbb{R}^{n \times d}\): 데이터 행렬 (n개 샘플, d개 특성)
\(\mathbf{y} \in \mathbb{R}^n\): 정답 벡터
\((\mathbf{X}^\top\mathbf{X})^{-1}\mathbf{X}^\top\): 의사역행렬(pseudo-inverse). \(n \gg d\)이면 계산 가능
주의: \((\mathbf{X}^\top\mathbf{X})\)가 역행렬이 없으면 정규화(Ridge) 필요: \((\mathbf{X}^\top\mathbf{X} + \lambda I)^{-1}\)

🧠 선형회귀 → 딥러닝으로의 확장

선형회귀를 이해하면 딥러닝으로 가는 길이 열립니다. 차이는 비선형성(활성화 함수)과 층(depth) 뿐입니다.

개념선형회귀딥러닝 (레이어 1개)딥러닝 (다층)
모델 \(\hat{y} = \mathbf{w}^\top\mathbf{x} + b\) \(\hat{y} = \sigma(\mathbf{w}^\top\mathbf{x} + b)\) \(f_L \circ \cdots \circ f_1(\mathbf{x})\)
손실함수 MSE BCE (이진분류) CE (다중분류)
최적화 해석적 해 (OLS) 경사하강법 역전파 + 경사하강법
비선형성 없음 sigmoid, ReLU 복합 활성화 함수
표현 능력 선형 패턴만 비선형 패턴 가능 매우 복잡한 패턴
🔑 선형회귀 = MSE 최소화 = MLE (정규분포 가정)

노이즈가 정규분포를 따른다고 가정하면: \(y = \mathbf{w}^\top\mathbf{x} + b + \epsilon,\; \epsilon \sim \mathcal{N}(0, \sigma^2)\)

\[ \hat{\mathbf{w}}_{MLE} = \arg\min_\mathbf{w} \sum_{i=1}^n (y_i - \hat{y}_i)^2 = \hat{\mathbf{w}}_{OLS} \]

MSE 최소화 ↔ 정규 노이즈 가정 하의 MLE. 선형회귀와 최대우도추정이 같은 답을 준다는 수학적 연결!

🧠 확인 문제

단순 선형회귀에서 최적 기울기 \(w^*\)를 구하는 공식은?

(a)   \(w^* = \bar{y} / \bar{x}\) (평균의 비율)
(b)   \(w^* = \text{Var}(X) / \text{Cov}(X, Y)\) (분자·분모 반대)
(c)   \(w^* = \text{Cov}(X, Y) / \text{Var}(Y)\) (분모가 Y의 분산)
(d)   \(w^* = \text{Cov}(X, Y) / \text{Var}(X)\) — x와 y가 함께 변하는 정도를 x의 퍼짐으로 정규화
