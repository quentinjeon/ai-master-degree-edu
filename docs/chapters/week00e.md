## 섹션 구성

### 섹션 1: 집합과 속함 — "어디에 있냐"를 수학으로

**아이콘**: `∈`
**제목**: `집합 기호 — ∈, ⊂, ∪, ∩`

**설명**: 집합은 "원소의 모임"입니다. 수학에서 "어떤 것이 어디에 속하는가"를 표현할 때 집합 기호를 씁니다.

**표 (집합 기호 번역표)**:

| 기호 | 읽는 법 | 의미 | 예시 |
|------|---------|------|------|
| ∈ | "원소이다", "에 속한다" | x가 집합 A의 원소 | x ∈ ℝ → "x는 실수" |
| ∉ | "원소가 아니다" | x가 집합 A에 없음 | -1 ∉ ℕ → "-1은 자연수 아님" |
| ⊂ | "부분집합이다" | A의 모든 원소가 B에도 있음 | ℕ ⊂ ℝ |
| ∪ | "합집합" | A 또는 B에 있는 원소 | A ∪ B |
| ∩ | "교집합" | A 이면서 B인 원소 | A ∩ B |
| ℕ | 자연수 집합 | {1, 2, 3, ...} | n ∈ ℕ → "n은 자연수" |
| ℝ | 실수 집합 | 모든 실수 | x ∈ ℝ → "x는 실수" |

**callout (info)**:
- 내용: 논문에서 "Let x ∈ ℝⁿ be the input vector"를 보면 "x는 n차원 실수 벡터"라는 뜻이다. ℝⁿ = n개의 실수로 이루어진 집합. [1, 2, 3]은 ℝ³의 원소다.

### 섹션 2: 한정사 — "모든"과 "어떤"

**아이콘**: `∀`
**제목**: `∀ (모든)과 ∃ (어떤) — 논문 정리에서 가장 많이 나오는 기호`

**설명**: 수학에서 "모든"과 "어떤"을 기호로 씁니다.

**수식 박스 1**:
- 레이블: `한정사 기호`
- 수식:
```
\forall x \in \mathbb{R},\ f(x) \geq 0
\quad\text{("모든 실수 x에 대해 f(x)는 0 이상이다")}

\exists x \in \mathbb{R}\ \text{s.t.}\ f(x) = 0
\quad\text{("0이 되는 x가 실수 중 하나 존재한다")}
```
- 주석:
  - `∀` (for all): "모든 ~ 에 대해"
  - `∃` (there exists): "~ 이 존재한다"
  - `s.t.` (such that): "~ 를 만족하는"
  - 논문에서 Assumption: "∀x ∈ 𝒳, ‖f(x)‖ ≤ L" → "모든 입력에 대해 f(x)의 크기가 L 이하"

**Level 0 연결**:
```
Level 0: "모든 학생의 점수는 0 이상이다"
Level 1: ∀s ∈ Students, score(s) ≥ 0
```

**callout (tip)**:
- 내용: ε-N 정의 (`∀ε > 0, ∃N s.t. n > N ⇒ |aₙ - L| < ε`)를 처음 보면 암호처럼 보인다. 하지만 번역하면 "아무리 작은 ε을 줘도 N이 존재해서 그 이후 모든 항이 L에서 ε 이내에 있다"이다. 기호만 알면 읽힌다.

### 섹션 3: 함수의 수학적 정의

**아이콘**: `f`
**제목**: `f: X → Y — 함수를 수학으로 정의하는 법`

**설명**: Level 0에서 `def predict(x): return w*x + b`로 함수를 만들었습니다. 수학에서는 이것을 `f: ℝ → ℝ, f(x) = wx + b`로 씁니다.

**표 (Python 코드 ↔ 수학 표기)**:

| Python | 수학 표기 | 읽는 법 |
|--------|----------|---------|
| `def f(x):` | `f: X → Y` | f는 X에서 Y로 가는 함수 |
| `return w*x + b` | `f(x) = wx + b` | f(x)의 값은 wx + b |
| `f(3)` | `f(3) = w·3 + b` | x=3일 때 f의 값 |
| `max(scores)` | `argmax f(x)` | f를 최대로 하는 x |
| `min(loss)` | `argmin L(θ)` | 손실 L을 최소로 하는 θ |

**수식 박스 2**:
- 레이블: `함수의 수학적 정의`
- 수식:
```
f: \mathbb{R}^n \to \mathbb{R}
\quad\text{where}\quad
f(\mathbf{x}) = \mathbf{w}^\top \mathbf{x} + b
```
- 주석:
  - `f: ℝⁿ → ℝ`: n개의 숫자를 입력받아 숫자 하나를 출력
  - `w⊤x`: 가중치 벡터와 입력 벡터의 내적 (weighted sum)
  - `b`: 편향 (기본값 오프셋)
  - 이것이 선형회귀, 로지스틱 회귀 등 모든 선형 모델의 공통 형태

---

### 섹션 4: Σ 표기법 — 합계를 수학으로

**아이콘**: `Σ`
**제목**: `Σ 표기 — 반복 합산을 한 줄로`

**설명**: Level 0에서 `sum(scores)`를 썼습니다. 수학에서는 `Σ`(시그마)를 씁니다.

**수식 박스 3**:
- 레이블: `Σ 표기법과 Python 대응`
- 수식:
```
\sum_{i=1}^{n} x_i = x_1 + x_2 + \cdots + x_n
\quad\Leftrightarrow\quad
\texttt{sum(x)}

\bar{x} = \frac{1}{n}\sum_{i=1}^{n} x_i
\quad\Leftrightarrow\quad
\texttt{np.mean(x)}
```
- 주석:
  - `i=1 ~ n`: i가 1부터 n까지 증가하면서 합산
  - `xᵢ`: i번째 원소
  - `x̄` (x bar): 평균

**개념 카드**:
- 제목: `🔑 Σ 읽는 법 4단계`
- 내용:
  1. 아래: 시작값 (i=1 → 1부터)
  2. 위: 끝값 (n → n까지)
  3. 뒤: 더할 것 (xᵢ → i번째 원소)
  4. 전체: i를 1~n까지 바꿔가며 xᵢ를 모두 더함

## 인터랙티브: y = wx + b 그래프 조절기

**제목**: `🔢 함수 시각화 — w와 b를 바꿔보세요`

**파라미터**:

| ID | 이름 | min | max | step | default |
|----|------|-----|-----|------|---------|
| `w_val` | 가중치 w | -3 | 3 | 0.1 | 1 |
| `b_val` | 편향 b | -5 | 5 | 0.5 | 0 |

**JavaScript 로직**:
```javascript
function updateLinear() {
  const w = parseFloat(document.getElementById('w_val').value);
  const b = parseFloat(document.getElementById('b_val').value);
  document.getElementById('w_val-v').value = w.toFixed(1);
  document.getElementById('b_val-v').value = b.toFixed(1);

  // x = -3, -2, -1, 0, 1, 2, 3
  const xs = [-3, -2, -1, 0, 1, 2, 3];
  const ys = xs.map(x => (w * x + b).toFixed(2));
  const table = xs.map((x, i) =>
    `x=${x}: f(x) = ${w.toFixed(1)}×${x} + ${b.toFixed(1)} = <strong>${ys[i]}</strong>`
  ).join('<br/>');

  const slope = w > 0 ? '우상향 ↗' : w < 0 ? '우하향 ↘' : '수평 →';
  const intercept = b > 0 ? `y절편: +${b.toFixed(1)}` : b < 0 ? `y절편: ${b.toFixed(1)}` : 'y절편: 0 (원점 통과)';

  document.getElementById('linear-res').innerHTML =
    `<strong>f(x) = ${w.toFixed(1)}x + ${b.toFixed(1)}</strong>
     &nbsp;|&nbsp; 기울기: ${slope} &nbsp;|&nbsp; ${intercept}<br/><br/>
     ${table}`;
}
updateLinear();
```

## 퀴즈

**질문**: 수학 표기 `f: ℝ² → ℝ`이 의미하는 것은?

| 선택지 | 정답 여부 |
|--------|----------|
| (a) 실수 2개를 출력하는 함수 | ❌ |
| (b) 자연수를 입력받아 실수를 출력하는 함수 | ❌ |
| (c) 실수 1개를 입력받아 2개를 출력하는 함수 | ❌ |
| (d) 실수 2개를 입력받아 실수 1개를 출력하는 함수 | ✅ |

**정답 위치**: (d)
**해설**: `ℝ²`는 실수 2개 (입력), `ℝ`는 실수 1개 (출력). 예: 키와 몸무게(2개)를 입력받아 건강점수(1개)를 출력하는 함수가 `f: ℝ² → ℝ`. 선형회귀도 `f: ℝⁿ → ℝ`이다.
