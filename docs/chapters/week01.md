## 섹션 구성

### 섹션 1: 집합 (Set)

**아이콘**: `∈`  
**제목**: `집합 (Set)이란?`

**설명**: 집합은 뚜렷이 구별되는 원소들의 모임입니다. AI 논문에서 데이터셋, 파라미터 공간, 클래스 레이블 등을 집합으로 표현합니다.

**수식 박스 1**:
- 레이블: `AI 논문에서의 집합 표기`
- 수식: `\mathcal{D} = \{(\mathbf{x}_1, y_1), (\mathbf{x}_2, y_2), \ldots, (\mathbf{x}_n, y_n)\}`
- 주석: `\mathcal{D}`: 훈련 데이터셋, `(\mathbf{x}_i, y_i)`: i번째 입력-출력 쌍

**표 (핵심 기호)**:

| 기호 | 읽는 법 | 의미 | 예시 |
|------|---------|------|------|
| `x ∈ A` | x는 A에 속한다 | x는 집합 A의 원소 | `3 ∈ {1,2,3}` |
| `A ⊆ B` | A는 B에 포함된다 | A는 B의 부분집합 | `{1,2} ⊆ {1,2,3}` |
| `A ∪ B` | A 합집합 B | A 또는 B의 원소 | `{1,2} ∪ {2,3} = {1,2,3}` |
| `A ∩ B` | A 교집합 B | A이면서 B인 원소 | `{1,2} ∩ {2,3} = {2}` |
| `∀x` | 모든 x에 대해 | for all x | `∀x ∈ ℝ, x²≥0` |
| `∃x` | 어떤 x가 있어서 | there exists x | `∃x : f(x)=0` |
| `ℝᵈ` | d차원 실수 공간 | d개 실수의 벡터 공간 | `x ∈ ℝ⁷⁶⁸` (BERT) |

### 섹션 2: 함수 (Function)

**아이콘**: `f`  
**제목**: `함수 (Function)`

**설명**: 함수 `f: X → Y`는 정의역 X의 각 원소를 공역 Y의 정확히 하나의 원소에 대응시키는 규칙입니다.

**개념 카드**:
- 제목: `🔑 핵심 개념 — 머신러닝 모델은 함수다`
- 내용: 모든 AI 모델은 수학적으로 함수다.
- 수식: `f_\theta : \mathcal{X} \to \mathcal{Y}`
- 설명: 파라미터 θ를 가진 모델이 입력 공간에서 출력 공간으로 매핑

**수식 박스 2**:
- 레이블: `딥러닝의 함수 합성`
- 수식: `f(\mathbf{x}) = f_L \circ f_{L-1} \circ \cdots \circ f_1(\mathbf{x})`
- 주석: L개 레이어 = L개 함수의 합성, `∘`는 합성 연산자

**callout (tip)**:
- 제목: `논문에서 자주 보이는 표현`
- 내용: `\argmin_\theta \mathcal{L}(\theta)` — "손실함수 ℒ을 최소화하는 파라미터 θ를 찾아라"
