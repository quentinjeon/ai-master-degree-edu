## 섹션 구성

### 섹션 1: 수열과 수렴의 직관

**아이콘**: `∞`  
**제목**: `수열이란 무엇인가?`

**설명**: 수열(sequence)은 자연수 인덱스로 배열된 수들의 목록입니다. `(aₙ)_{n=1}^∞` 처럼 씁니다. 수열이 수렴한다는 것은 항이 특정 값 L에 "충분히 가까워진다"는 것이지만, "충분히"를 정확히 정의하는 것이 핵심입니다.

**표 (핵심 수열 예시)**:

| 수열 | 수렴 여부 | 극한값 | 논문 연결 |
|------|----------|--------|----------|
| `1/n` | ✅ 수렴 | 0 | 학습률 감소 스케줄 |
| `(-1)ⁿ` | ❌ 발산 | 없음 | 발산하는 학습 |
| `(1 + 1/n)ⁿ` | ✅ 수렴 | e ≈ 2.718 | 자연상수 정의 |
| `aₙ₊₁ = aₙ - α∇f(aₙ)` | 조건부 | f의 최솟값 | 경사하강법 |

### 섹션 2: ε-N 정의 (수열의 극한)

**아이콘**: `ε`  
**제목**: `극한의 엄밀한 정의 — ε-N 논증`

**설명**: 수열 `(aₙ)`이 L로 수렴한다는 것은 다음을 의미합니다: 아무리 작은 양수 ε을 주어도, 충분히 큰 N이 존재해서 n > N이면 항상 |aₙ - L| < ε이 된다.

**수식 박스 1**:
- 레이블: `수열 극한의 ε-N 정의`
- 수식:
```
\lim_{n \to \infty} a_n = L
\iff
\forall \varepsilon > 0,\ \exists N \in \mathbb{N}\ \text{s.t.}\ n > N \Rightarrow |a_n - L| < \varepsilon
```
- 주석:
  - `∀ε > 0`: 어떤 작은 ε을 골라도 (상대방이 아무리 까다롭게 골라도)
  - `∃N`: 충분히 큰 출발점 N이 **존재한다**
  - `n > N ⇒ |aₙ - L| < ε`: 그 이후 모든 항이 L의 ε-근방에 있다
  - 이 정의로 "수렴"이 수학적 증명 가능한 명제가 된다

**callout (tip)**:
- 내용: **ε은 상대방이 고르고, N은 내가 고른다.** "얼마나 가까워야 하냐"(ε)를 먼저 받고, 그에 맞는 N을 제시하는 게임이다. 논문에서 "for any ε > 0, there exists N such that..."은 이 정의를 쓴 것이다.

### 섹션 3: Cauchy 수열

**아이콘**: `🔗`  
**제목**: `Cauchy 수열 — 극한값을 몰라도 수렴을 증명하는 법`

**설명**: 극한값 L을 알지 못해도 수렴을 증명할 수 있습니다. Cauchy 수열은 항들끼리 서로 충분히 가까워지면 됩니다. 완비 공간(ℝ 등)에서 Cauchy 수열 ⟺ 수렴 수열입니다.

**수식 박스 2**:
- 레이블: `Cauchy 수열의 정의`
- 수식:
```
(a_n) \text{ is Cauchy}
\iff
\forall \varepsilon > 0,\ \exists N\ \text{s.t.}\ m, n > N \Rightarrow |a_m - a_n| < \varepsilon
```
- 주석:
  - 극한값 L이 필요 없다 — 항들끼리의 거리만 본다
  - ℝ(실수)은 완비(complete): 모든 Cauchy 수열이 수렴한다
  - AI 연결: SGD 반복이 Cauchy 수열처럼 수렴함을 보이는 게 수렴 증명의 핵심

**개념 카드**:
- 제목: `🔑 완비성 (Completeness) — 왜 중요한가`
- 내용: ℝ은 완비 공간이다 = Cauchy 수열은 반드시 수렴한다. 이것이 실수 해석학의 근간. 유리수 ℚ는 완비가 아니다 (√2로 수렴하는 유리수열이 ℚ 안에서 극한값이 없음). ML 이론에서 함수 공간의 완비성이 중요하게 등장한다.

### 섹션 4: 논문에서의 수렴 표기

**아이콘**: `📄`  
**제목**: `AI 논문에서의 수렴 표기법`

**표 (자주 등장하는 수렴 표기)**:

| 표기 | 읽는 법 | 의미 |
|------|---------|------|
| `aₙ → L` | aₙ이 L로 수렴 | lim aₙ = L |
| `aₙ = O(1/n)` | aₙ은 O(1/n) | c/n보다 빠르게 0에 수렴 |
| `aₙ = o(1)` | aₙ은 o(1) | aₙ → 0 (little-o) |
| `‖θₜ - θ*‖ ≤ C/√T` | 오차 경계 | T번 후 최적해 θ* 로부터의 거리 |
| `lim_{t→∞} f(θₜ) = f*` | 수렴 | 목적함수가 최솟값으로 수렴 |

**callout (info)**:
- 내용: 논문에서 "Theorem 1 (Convergence): Under Assumption A1-A3, the iterates satisfy ‖θₜ - θ*‖ ≤ C/√T"처럼 쓰인다. 이 T는 반복 횟수, C는 상수다. 이게 바로 경사하강법 수렴 속도를 ε-N 언어로 표현한 것.

## 인터랙티브: ε-N 수렴 시각화

**제목**: `🔢 수렴 시각화 — ε을 조절하면 N이 바뀐다`

**파라미터**:

| ID | 이름 | min | max | step | default |
|----|------|-----|-----|------|---------|
| `eps` | 허용 오차 ε | 0.001 | 0.5 | 0.001 | 0.1 |

**JavaScript 로직**:
```javascript
function updateEps() {
  const eps = parseFloat(document.getElementById('eps').value);
  document.getElementById('eps-v').value = eps.toFixed(3);

  // 수열 aₙ = 1/n, 극한 L = 0
  // |aₙ - 0| = 1/n < ε  →  n > 1/ε  →  N = ceil(1/ε)
  const N = Math.ceil(1 / eps);
  const aN = (1 / N).toFixed(5);
  const aN1 = (1 / (N + 1)).toFixed(5);

  document.getElementById('eps-res').innerHTML =
    `수열 aₙ = 1/n, 극한 L = 0<br/>
     ε = ${eps.toFixed(3)} 일 때, <strong>N = ${N}</strong><br/>
     a_${N} = 1/${N} = ${aN}  →  |a_${N} - 0| = ${aN}  ${parseFloat(aN) < eps ? '< ε ✅' : '≥ ε'}<br/>
     a_${N+1} = 1/${N+1} = ${aN1}  →  |a_${N+1} - 0| = ${aN1}  ${parseFloat(aN1) < eps ? '< ε ✅' : '≥ ε'}<br/>
     <span style="font-size:.78rem;color:var(--muted)">
       ε을 작게 하면 N이 커진다 — 더 많은 항이 지나야 수렴 조건을 만족한다
     </span>`;
}
updateEps();
```
