# 레벨 프레임워크 — 학습자 목표 단계 정의

> **버전**: 1.0.0  
> **작성일**: 2026-04-19  
> **목적**: 3단계 학습 목표를 명확히 정의하여 챕터 개발 방향을 통일한다

---

## 핵심 철학

> "역순 설계(Backward Design) — 최종 목표(레벨 3)에서 역으로 내려와 각 레벨의 완성 기준을 정의한다."

레벨 3의 완성 = **SCI급 AI 논문을 스스로 설계하고 쓸 수 있는 Stanford/MIT 수준**  
이 목표를 역순으로 분해하면 레벨 2, 레벨 1, 그리고 그 전의 레벨 0이 도출된다.

---

## 레벨 정의 (상세)

### ⚪ 레벨 0 — 사전 준비 (Pre-beginner)

**완성 기준**: AI를 이해할 수 있는 **사고 구조**를 만든다 (입문을 시작할 수 있는 상태)

| 항목 | 내용 |
|------|------|
| **독자 수준** | 초등학생 ~ 중학생, 수학/코딩 경험 없는 완전 입문자 |
| **목표** | 조건/반복/변수/함수의 구조적 사고, Python 기초, "입력→결과" 모델 직관 |
| **판단 기준** | Python으로 간단한 예측 프로그램을 혼자 만들 수 있다 |
| **한계** | SCI 논문 수식은 읽지 못함. 수학 기호 체계를 모름 |
| **기간** | 약 1년 (초등학생 수준에서 시작 시) |

**커리큘럼 구조**: 1년 4단계 (상세 내용: `docs/LEVEL0_CURRICULUM.md`)

| 단계 | 내용 | 도구 |
|------|------|------|
| 1~3개월 | 사고력 + 기초 수학 + Scratch | Scratch |
| 4~6개월 | Python + 데이터 개념 | Python |
| 7~9개월 | AI 개념 진입 (y=wx+b, 분류, 확률 직관) | Python |
| 10~12개월 | 미니 AI 프로젝트 3개 | Python |

**Level 0 → Level 1 브릿지 챕터** (레벨 0 완료 후 필수):

| 브릿지 챕터 | 내용 | 해결하는 갭 |
|------------|------|-----------|
| `Ch.00D` | Python 데이터 과학 도구 (numpy/pandas/matplotlib) | 코딩 도구 갭 |
| `Ch.00E` | 수학 브릿지 (∀, ∃, Σ, f: X→Y) | 수학 기호 갭 |
| `Ch.00F` | ML 실습 입문 (scikit-learn, 모델 평가) | 실습 경험 갭 |

**현황**: 📋 커리큘럼 설계 완료, 브릿지 챕터 명세 작성 완료 (2026-04-19)

### 🟢 레벨 1 — 입문 (Beginner)

**완성 기준**: SCI급 AI 논문을 읽고 이해할 수 있다

| 항목 | 내용 |
|------|------|
| **독자 수준** | 수학 배경이 약한 AI/ML 개발자, 대학원 준비생 |
| **목표** | 논문의 수식, 기호, 논리 흐름을 막힘 없이 따라갈 수 있다 |
| **판단 기준** | NeurIPS/ICML 논문의 Related Work + Method 섹션을 혼자 읽을 수 있다 |
| **한계** | 이해는 하지만 저자의 가정/선택에 의문을 제기하기 어렵다 |

**해당 챕터**: Phase 1 (Ch.01 ~ Ch.14)

| 챕터 범위 | 내용 |
|-----------|------|
| Ch.00a~00c | 로그·지수, 수열·Σ, 경우의 수 (사전 준비) |
| Ch.01 | 집합과 함수 |
| Ch.02~02b | 미분과 최적화, 적분 기초 |
| Ch.03~03b | 선형대수·행렬미분, 선형회귀 |
| Ch.04 | 확률론·베이즈·독립성 |
| Ch.05~06 | 확률변수·분포, 기대값·분산 |
| Ch.07~08 | 표본·MLE, MAP·Bias-Variance |
| Ch.09~10 | Logistic Regression, Softmax |
| Ch.11 | 정보이론·KL Divergence |
| Ch.12~13 | 딥러닝·LLM, Calibration |
| Ch.09b | 모델 평가지표 |
| Ch.14 | 논문 읽기·쓰기 |

**현황**: ✅ Phase 1 전 챕터 QA_PASS 완료 (2026-04-19)

### 🔵 레벨 2 — 중급 (Intermediate)

**완성 기준**: SCI급 AI 논문을 완전히 이해하고, 저자의 가정에 논리적으로 질문을 던질 수 있다

| 항목 | 내용 |
|------|------|
| **독자 수준** | 대학원생, 논문 리뷰를 준비하는 연구자 |
| **목표** | 논문의 Theorem/Proof를 따라가고, "왜 이 가정이 필요한가?", "이 증명의 약점은 무엇인가?"를 묻는다 |
| **판단 기준** | NeurIPS/ICML 논문 리뷰어로서 명확한 질문과 약점을 기술할 수 있다 |
| **한계** | 새로운 논문을 직접 설계하거나 실험 체계를 구성하기는 어렵다 |

**해당 챕터**: Phase 2 (Ch.15 ~ Ch.28 예정)

| 챕터 범위 | 카테고리 | 내용 |
|-----------|----------|------|
| Ch.15~17 | Phase 2-A · 해석학 기초 | ε-δ, Lipschitz 연속, 적분·테일러 급수 |
| Ch.18~20 | Phase 2-B · 볼록 최적화 | 볼록 집합, Lagrangian·KKT, 수렴 속도 |
| Ch.21~23 | Phase 2-C · 측도 기반 확률론 | σ-대수·확률공간, 수렴 개념, 중심극한정리 증명 |
| Ch.24~28 | Phase 2-D · 고급 ML 이론 | PAC 학습, Rademacher 복잡도, 가우시안 프로세스 (계획 중) |

**현황**: ✅ Phase 2 전 챕터 QA_PASS 완료 (2026-04-19)

### 🔴 레벨 3 — 고급 (Advanced)

**완성 기준**: SCI급 AI 논문을 직접 설계하고 쓸 수 있는 Stanford/MIT 수준

| 항목 | 내용 |
|------|------|
| **독자 수준** | 박사과정, 시니어 연구자 |
| **목표** | 새로운 문제를 정의하고, 수학적 프레임워크를 설계하고, Theorem을 증명하고, 실험을 구성하여 논문으로 제출한다 |
| **판단 기준** | NeurIPS/ICML에 1저자 논문 제출 및 통과 가능 수준 |
| **핵심 역량** | 연구 방법론 설계, Ablation Study, 새로운 Loss/목적함수 설계, 수학적 증명 작성, 리뷰어 수준의 논리적 반박 대응 |

**해당 챕터**: Phase 3 (계획 중)

| 챕터 범위 | 내용 (예정) |
|-----------|-------------|
| Ch.29~32 | 연구 방법론 설계 (실험 설계, Ablation, Baseline 선택) |
| Ch.33~36 | 수학적 증명 작성법 (Lemma → Theorem → Corollary 구조) |
| Ch.37~40 | 논문 구조 설계 (Introduction의 기여 설명, Limitation 작성) |
| Ch.41~44 | 리뷰 대응 (Rebuttal 작성, 약점 보완 실험) |

**현황**: ✅ Phase 3 전 챕터 QA_PASS 완료 (2026-04-19, Ch.27~Ch.40)

## 참고: 레벨 간 연결 콘텐츠 규칙

레벨 2 챕터에서는 레벨 1에서 배운 내용과의 연결을 반드시 명시한다.

예시 (Ch.15 수열과 극한):
- "레벨 1에서 경사하강법이 '수렴한다'고 배웠다. 레벨 2에서는 그 '수렴'의 수학적 의미를 ε-N 정의로 엄밀하게 이해한다."

이를 통해 학습자가 레벨 간 단절감 없이 이어서 공부할 수 있도록 한다.
