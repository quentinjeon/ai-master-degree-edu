### Day 51~60 — 학습 원리: 경사하강법 코드 구현 ← ❌ 기존 챕터 부족

**관련 챕터**: `ch02.html` (개념) + **신규 필요: `ch02d.html`** (구현 실습)

> ⚠️ Ch.02는 경사하강법 시뮬레이터(슬라이더)를 제공하지만,
> `θ = θ - η∇J(θ)`를 **직접 numpy로 구현하는 코드 체크리스트**가 없다.

| 체크 | 개념 항목 | 챕터 |
|------|----------|------|
| ☐ | "오차 줄이는 방향으로 w를 조금씩 이동" 설명 가능 | Ch.02 |
| ☐ | gradient = 기울기 = "어느 방향으로 얼마나 이동할지" 설명 가능 | Ch.02 |
| ☐ | 학습률 η가 너무 크면/작으면 어떻게 되는지 설명 가능 | Ch.02 |
| ☐ | `θ = θ - η * gradient` 코드 구현 가능 | **Ch.02D** |
| ☐ | 반복 학습 루프 구현 (`for epoch in range(100)`) | **Ch.02D** |

**코드 체크 (신규 챕터 Ch.02D)**

| 체크 | 코드 항목 |
|------|----------|
| ☐ | `gradient = 2/n * np.dot(X.T, (y_pred - y))` — gradient 계산 |
| ☐ | `w = w - lr * gradient` — 파라미터 업데이트 |
| ☐ | `for epoch in range(1000): ...` — 반복 학습 |
| ☐ | 에폭마다 loss 출력 → 감소하는 것 확인 |
| ☐ | 학습률 바꿔서 수렴 속도 차이 관찰 |

**완료 기준**: 합성 데이터로 경사하강법 scratch 구현 후 loss 감소 확인
