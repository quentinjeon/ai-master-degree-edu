## 7~9개월: AI 개념 진입

### 목표
- "입력 → 결과" 모델 이해
- 간단한 예측 구현

### 7개월차 (함수 기반 사고)

**핵심 개념**: `y = w * x + b`

- `x`: 입력 (키)
- `w`: 가중치 (영향력 크기)
- `b`: 편향 (기본값)
- `y`: 출력 (예측 몸무게)

**학습 내용**
- 입력값에 따라 결과 변화
- 가중치 개념 (w가 크면 영향이 크다)
- 직접 w, b 값 바꿔서 실험

**주차별 실행**

| 주차 | 학습 | 실습 |
|------|------|------|
| 1주 | y = wx + b 직관 | 키 170 → w=0.5, b=10 → 몸무게? |
| 2주 | 여러 w 값 실험 | w가 달라지면 예측이 어떻게 변하나 |
| 3주 | 오차 개념 | 예측 - 실제 = 틀림의 크기 |
| 4주 | 프로그램 완성 | 키 → 몸무게 예측기 |

**결과물**: 키 → 몸무게 예측 프로그램

---

- 조건 기반 분류 (if/elif 활용)

**결과물**: 동물 분류기 (크기, 색 기반)

### 9개월차 (확률 느낌)

**핵심 개념**: `P(A|B)` — B가 주어졌을 때 A일 확률

- 직관으로만 접근: "어제 흐렸을 때 70%는 비가 왔었다"
- 빈도를 세서 확률 계산

**학습 내용**
- "가능성" 개념 (0~100%)
- 빈도 기반 판단 (많이 일어난 것이 더 가능성 높다)
- 데이터로 확률 추정

**주차별 실행**

| 주차 | 학습 | 실습 |
|------|------|------|
| 1주 | 빈도 세기 | 100번 중 비 온 날 세기 |
| 2주 | 조건부 빈도 | 흐린 날 중 비 온 날 비율 |
| 3주 | 딕셔너리로 저장 | `{"흐림": {"비": 70, "맑음": 30}}` |
| 4주 | 날씨 예측기 완성 | 오늘 구름 → 내일 비 확률 |

**결과물**: 날씨 맞추기 (간단 확률)

### 11개월차: 간단 예측 모델

**프로젝트**: 공부시간 → 점수 예측

**로직**:
```python
# 7개월차에서 배운 y = wx + b 활용
def predict_score(hours):
    w = 8   # 1시간 공부하면 8점 상승 (직접 설정)
    b = 40  # 기본 점수
    return w * hours + b
```

**주차별 실행**

| 주차 | 작업 |
|------|------|
| 1주 | 실제 데이터 모으기 (공부시간, 점수) |
| 2주 | w, b 수동으로 맞추기 (오차 최소화) |
| 3주 | 예측 vs 실제 비교 출력 |
| 4주 | 완성 + 그래프 (텍스트로) |

**결과물**: 공부시간 → 점수 예측기
